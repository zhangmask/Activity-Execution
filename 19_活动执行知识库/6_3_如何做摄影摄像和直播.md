# 6_3 如何做摄影摄像和直播

> 适用场景：活动全程需要摄影摄像记录，并可能需要进行线上直播，让无法到场的人也能参与。
> 前置条件：[6_2_如何运营社交媒体](6_2_如何运营社交媒体.md)

---

## 第一步：制定拍摄清单

### 按环节拍摄清单

| 环节 | 必拍内容 | 拍摄要求 | 拍摄人 |
|---|---|---|---|
| **签到** | 签到现场全景、签到特写、物料包发放 | 广角+特写 | 摄影A |
| **开幕式** | 全场全景、嘉宾发言、观众反应、主持人 | 固定机位+游走 | 摄影A+B |
| **开赛/开发** | 选手工作场景、团队讨论、导师指导 | 游走抓拍 | 摄影A |
| **展区** | 展位全景、展品特写、观众互动 | 广角+特写 | 摄影B |
| **路演/展示** | 路演全景、选手展示、评委提问、观众反应 | 固定+游走 | 摄影A+B |
| **评分/投票** | 评委打分、游客投票、投票箱 | 特写 | 摄影B |
| **颁奖** | 获奖者上台、颁奖嘉宾颁奖、合影、表情特写 | 固定+游走 | 摄影A+B |
| **花絮** | 后勤保障、志愿者工作、观众互动、场地空镜 | 随时抓拍 | 全员 |
| **合影** | 大合影、小组合影、自拍 | 广角 | 摄影A |

### 拍摄Tips

| 场景 | 建议 |
|---|---|
| 室内活动 | 提高ISO、注意白平衡、避免逆光 |
| 舞台/讲台 | 从正面45度角拍摄，避免正面平拍 |
| 选手工作 | 抓拍专注表情、团队讨论、屏幕画面 |
| 路演展示 | 拍演讲者+观众反应的交替镜头 |
| 合影 | 确保所有人入镜、提醒看镜头、连拍3-5张 |
| 花絮 | 自然抓拍，不要摆拍 |

## 第二步：机位安排

### 单机位方案（小型活动）

```
                  [舞台/展示区]
                      
        📷 固定机位（正前方偏右45°）
        
              [观众席]
```

- 1台摄像机固定拍摄舞台
- 1名摄影师全场游走拍摄

### 双机位方案（中型活动，推荐）

```
                  [舞台/展示区]
                      
    📷 机位A              📷 机位B
    （正前方偏左45°）      （侧面全景）
    
              [观众席]
              
         🚶 游走摄影师C
```

- 机位A：舞台正面偏左45°，拍演讲者特写
- 机位B：侧面全景位，拍舞台全貌和观众
- 游走摄影师：拍摄花絮、观众反应、展品

### 三机位方案（大型活动）

```
                  [舞台/展示区]
                      
    📷 机位A         📷 机位B         📷 机位C
    （正前方）       （侧面）         （观众后方）
    
              [观众席]
              
         🚶 游走摄影师D
```

- 机位A：正前方，拍摄舞台全景（兼直播主机位）
- 机位B：侧面，拍摄演讲者特写
- 机位C：观众后方，拍摄观众全景
- 游走摄影师：花絮和抓拍

### 机位检查 Checklist

- [ ] 每个机位视野确认（无遮挡）
- [ ] 三脚架稳定、高度合适
- [ ] 相机电池充满、存储卡充足
- [ ] 拍摄参数设置（分辨率、帧率、白平衡）
- [ ] 音频录制方案确认（机内收音/外接麦克风）
- [ ] 机位之间不会互相影响

## 第三步：直播设备清单

### 基础直播设备

| 设备 | 数量 | 用途 | 推荐型号/规格 | 预算参考 |
|---|---|---|---|---|
| 摄像机/相机 | 1-2台 | 画面采集 | 有HDMI输出的相机或摄像机 | 0-5000元（可租） |
| 采集卡 | 1个 | 将相机画面输入电脑 | 圆刚/天创恒达 | 200-800元 |
| 笔记本电脑 | 1台 | 运行OBS推流 | i5以上、8G内存 | — |
| 三脚架 | 1-2个 | 固定相机 | 稳定、高度可调 | 100-300元 |
| HDMI线 | 2条 | 连接相机和采集卡 | 3-5米 | 30-50元 |
| 无线麦克风 | 1-2个 | 收音 | 领夹麦或指向性麦 | 200-1000元 |
| 音频线 | 1条 | 连接麦克风和电脑/相机 | 3.5mm/XLR | 20-50元 |
| 移动WiFi/4G网卡 | 1个 | 保证网络稳定 | 4G/5G随身WiFi | 100-300元 |
| 延长线/插线板 | 2-3个 | 供电 | 带过载保护 | 50-100元 |

### 进阶直播设备（可选）

| 设备 | 用途 | 说明 |
|---|---|---|
| 导播台 | 多机位切换 | 专业直播用，可租 |
| 提词器 | 嘉宾/主持人看稿 | 可用平板替代 |
| 补光灯 | 环境光线不足时补光 | LED面板灯 |
| 监视器 | 实时监看直播画面 | 可用iPad替代 |
| 硬件编码器 | 替代OBS，更稳定 | 预算充足时考虑 |

## 第四步：OBS设置指南

### OBS基础设置

#### 1. 下载安装

- 官网下载：https://obsproject.com
- 安装后打开，首次运行会弹出"自动配置向导"，选择"手动配置"

#### 2. 视频设置

| 设置项 | 推荐值 |
|---|---|
| 基础分辨率 | 1920 × 1080 |
| 输出分辨率 | 1920 × 1080（或1280 × 720） |
| 帧率 | 30 FPS |

#### 3. 输出设置

| 设置项 | 推荐值 |
|---|---|
| 输出模式 | 高级 |
| 编码器 | x264（软件编码）或 NVENC（N卡硬件编码） |
| 码率 | 2500-4000 Kbps（1080P）/ 1500-2500 Kbps（720P） |
| 关键帧间隔 | 2秒 |
| CPU预设 | veryfast（平衡性能和质量） |

#### 4. 音频设置

| 设置项 | 推荐值 |
|---|---|
| 采样率 | 48 kHz |
| 声道 | 立体声 |
| 音频码率 | 128-160 Kbps |

### OBS推流设置

1. 打开"设置" → "推流"
2. 服务选择：选择直播平台（B站/抖音/YouTube等）
3. 如果平台不在列表中，选择"自定义"
4. 填入平台提供的"推流地址"和"推流密钥"
5. 点击"确定"保存

### 直播前测试 Checklist

- [ ] 画面：采集卡/摄像头画面正常显示
- [ ] 音频：麦克风收音正常，音量合适
- [ ] 网络：上行带宽至少 10Mbps（测速确认）
- [ ] 推流：推流测试成功（画面在直播平台能正常看到）
- [ ] 字幕/Logo：叠加层显示正常
- [ ] 场景切换：各场景切换流畅
- [ ] 电池/电源：所有设备供电稳定
- [ ] 备用方案：如果直播中断的应急处理

## 第五步：素材管理规范

### 素材命名规范

```
格式：[日期]_[活动]_[环节]_[序号]

示例：
20260615_历史活动_开幕式_001.jpg
20260615_历史活动_路演_015.jpg
20260615_历史活动_颁奖_003.mp4
```

### 素材分类存储

```
素材文件夹/
├── 01_照片/
│   ├── 01_签到/
│   ├── 02_开幕式/
│   ├── 03_开发/
│   ├── 04_展区/
│   ├── 05_路演/
│   ├── 06_颁奖/
│   └── 07_花絮/
├── 02_视频/
│   ├── 01_开幕式完整录像/
│   ├── 02_路演完整录像/
│   ├── 03_颁奖完整录像/
│   └── 04_花絮视频/
├── 03_直播录像/
│   └── [按日期和场次]
└── 04_原始素材/
    └── [未处理的原始文件]
```

### 素材管理 Checklist

- [ ] 每天活动结束后1小时内，将所有素材备份到电脑
- [ ] 素材按命名规范重命名
- [ ] 素材按分类存入对应文件夹
- [ ] 原始素材单独保留一份（不覆盖）
- [ ] 至少备份2份（电脑+移动硬盘/云盘）
- [ ] 当天筛选出精选照片（20-30张），用于当晚宣发
- [ ] 活动结束后3天内，整理全部素材并归档

### 精选照片筛选标准

| 标准 | 说明 |
|---|---|
| 画面清晰 | 不模糊、不过曝 |
| 构图合理 | 主体突出、背景干净 |
| 表情自然 | 笑脸、专注、投入 |
| 信息量大 | 能体现活动氛围和亮点 |
| 可用性强 | 可用于宣传、回顾、社交媒体 |

## 第六步：人员配置

### 摄影摄像团队配置

| 岗位 | 人数 | 职责 | 设备 |
|---|---|---|---|
| 主摄影师 | 1人 | 负责舞台/主会场拍摄 | 相机+三脚架 |
| 副摄影师 | 1人 | 负责花絮、展区、观众 | 相机+手持稳定器 |
| 摄像师 | 1人 | 负责视频录制/直播 | 摄像机+三脚架 |
| 直播运营 | 1人 | OBS操作、场景切换、弹幕互动 | 笔记本电脑 |

> 小型活动可精简为2人：1摄影+1摄像兼直播。

## 常见问题

**Q：没有专业摄影师怎么办？**
A：①邀请有摄影爱好的朋友帮忙；②在志愿者中招募有摄影经验的人；③找摄影社团合作（提供免费参加活动的机会换拍摄）；④最简方案：用手机拍摄，确保光线充足、画面稳定。

**Q：直播画面卡顿怎么办？**
A：①降低输出分辨率（1080P→720P）；②降低码率（4000→2000 Kbps）；③检查网络上行带宽（至少10Mbps）；④关闭其他占用网络的程序；⑤如果WiFi不稳定，改用4G网卡。

**Q：直播音频有杂音怎么办？**
A：①使用领夹麦克风（离嘴近、收音清晰）；②避免使用笔记本内置麦克风；③在OBS中设置噪音抑制滤镜；④远离空调、风扇等噪音源。

**Q：存储卡满了怎么办？**
A：①每个摄影师至少准备2张存储卡；②活动间歇及时导出素材到电脑；③提前检查存储卡容量是否足够（1小时1080P视频约需10-15GB）。

**Q：照片/视频涉及隐私问题怎么处理？**
A：①在签到处告知参与者"活动将拍摄记录"；②拍摄时避免拍到不愿入镜的人；③发布前检查照片中是否有不当内容；④如果有人要求删除其照片，及时配合处理。

## 历史教训

> 历史活动的经验：摄影摄像是活动最重要的记录方式，建议至少安排2名摄影师（1固定+1游走）和1名摄像师。直播设备提前1天测试，特别是网络和音频。OBS场景要提前配置好（至少3个场景：主讲、PPT+画中画、等待画面），场景切换要提前演练。素材管理非常重要，每天活动结束后立即备份，按规范命名和分类。精选照片在活动当晚就要筛选出来，用于当晚的宣发推送。如果团队没有专业摄影经验，建议优先保证"拍到"而非"拍好"，手机拍摄也比不拍要好得多。
