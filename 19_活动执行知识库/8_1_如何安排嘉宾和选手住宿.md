# 如何安排嘉宾和选手住宿

> **适用场景**：hackathon活动涉及外地嘉宾或选手参加，需要统一安排住宿。
> **前置条件**：已完成嘉宾/选手名单确认（参考 3_1_嘉宾管理与邀请）。

---

### 需求收集要点

- **身份分类**：嘉宾、选手、工作人员、志愿者——不同身份的住宿标准不同
- **房型需求**：大床房、标间、拼住——根据预算和身份确定
- **特殊需求**：无烟房、楼层偏好、过敏信息、无障碍房间
- **收集截止时间**：建议在活动前2周截止，留出预订和调整时间

## 第二步：选择酒店

### 酒店选择标准

| 评估维度 | 嘉宾酒店标准 | 选手酒店标准 |
|----------|-------------|-------------|
| 距离场地 | 步行10分钟或车程15分钟内 | 车程30分钟内 |
| 房间类型 | 大床房/行政房 | 标间/多人间 |
| 价格区间 | 300-600元/晚 | 100-250元/晚 |
| 早餐 | 含早餐 | 含早餐优先 |
| WiFi | 必须稳定 | 必须稳定 |
| 会议室 | 有会议室/休息区 | 有公共区域即可 |
| 停车场 | 有停车位 | 不强求 |

### 酒店选择流程

1. **确定需求**：人数、房型、日期、预算
2. **筛选候选**：在携程/美团/飞猪上搜索活动场地附近的酒店
3. **电话询价**：直接联系酒店销售部，说明团体预订需求（通常10间以上有团体价）
4. **实地考察**：如果可能，提前去看房间和公共区域
5. **签订协议**：确认房价、早餐、取消政策，签订住宿协议

## 第三步：安排入住退房

### 入住流程

1. **提前分房**：按名单分配房间号，制作入住名单表
2. **酒店对接**：提前1天与酒店确认房间数量和房型
3. **办理入住**：
   - 现场设签到处，核对身份信息
   - 发放房卡 + 住宿须知（含WiFi密码、早餐时间、紧急联系人）
   - 签收登记
4. **住宿须知内容**：
   - WiFi名称和密码
   - 早餐时间和地点
   - 活动场地交通指引
   - 紧急联系人电话
   - 退房时间和流程

### 退房流程

1. **退房提醒**：退房前一晚群内通知退房时间和流程
2. **房卡回收**：在退房点统一回收房卡
3. **费用结算**：与酒店核对实际入住天数和费用
4. **押金退还**：确认押金退还方式和时间

## 第四步：安排接站

### 接站需求收集表

| 姓名 | 身份 | 到达时间 | 交通方式 | 到达站点 | 联系方式 | 是否需要接站 |
|------|------|----------|----------|----------|----------|-------------|
| 张三 | 嘉宾 | 6/15 14:00 | 高铁 | XX站 | 138xxxx | 是 |
| 李四 | 选手 | 6/15 10:30 | 飞机 | XX机场 | 139xxxx | 否（自行前往） |

### 接站安排要点

- **优先接嘉宾**：嘉宾的接站是必须的，体现主办方的重视
- **接站人员**：安排志愿者或工作人员，手持接站牌
- **接站牌模板**：`欢迎 [嘉宾姓名] 老师 | [活动名称]`
- **车辆安排**：根据人数安排网约车或包车
- **路线规划**：提前规划从车站/机场到酒店的最优路线
- **实时沟通**：到达前2小时确认航班/列车是否准点

## 常见问题

**Q：选手住宿费用谁来承担？**
A：一般有三种模式：①主办方全额承担（预算充足时）；②选手自费，主办方谈团体价（最常见）；③主办方补贴一部分。建议在报名时就说明住宿政策。

**Q：酒店要求预付定金怎么办？**
A：正常流程，一般预付1-2晚房费作为定金。确认好取消政策（提前几天可免费取消），保留付款凭证。

**Q：临时有嘉宾行程变更怎么办？**
A：提前与酒店确认"加房/减房"政策。一般提前3天可以免费调整，当天取消可能会扣费。

**Q：男女选手如何安排拼住？**
A：严格按性别分开安排拼住，绝对不能混住。提前在需求收集时确认性别信息。

## 历史教训

在hackathon活动中，住宿安排建议：
- **通宵hackathon**：如果是48小时通宵活动，选手可能不需要酒店，但嘉宾和导师仍需安排
- **场地过夜方案**：有些hackathon提供场地过夜（睡袋+气垫床），成本更低但体验一般
- **嘉宾住宿**：嘉宾住宿是基本礼仪，建议安排品质较好的酒店，体现尊重
- **选手拼住**：建议2人一间标间，在报名时就收集拼住偏好（是否介意拼住、是否有指定室友）
- **住宿群**：建立住宿专属群，方便统一通知和答疑
