# 变量与占位符说明

> 本知识库使用统一的变量和占位符，方便其他团队根据实际情况填写。

---

## 时间变量

| 变量 | 含义 | 示例 |
|------|------|------|
| **T日** | 活动第一天 | 如果活动是5月1日，则T日=5月1日 |
| **T-N天** | 活动前N天 | T-30天=活动前30天 |
| **T+N天** | 活动后N天 | T+7天=活动后7天 |

### 常用时间点

| 变量 | 含义 | 说明 |
|------|------|------|
| T-60天 | 筹备启动期 | 开始组建团队、确定方向 |
| T-45天 | 框架搭建期 | 场地确认、预算编制 |
| T-30天 | 外联资源期 | 赞助商对接、嘉宾邀请 |
| T-20天 | 细化落地期 | 物料设计、志愿者招募 |
| T-10天 | 冲刺执行期 | 物料制作、最终确认 |
| T-3天 | 最终彩排 | 全流程彩排、应急预案 |
| T日 | 活动执行日 | 活动正式开始 |
| T+1天 | 收尾期 | 物料回收、费用结算 |
| T+7天 | 复盘期 | 复盘会议、总结报告 |

## 使用示例

### 示例1：填写嘉宾邀请函

**原文**：
```
尊敬的 [嘉宾姓名]：
您好！我们诚挚地邀请您出席 **[活动名称]**。
活动时间：[开始时间] 至 [结束时间]
活动地点：[详细地址]
```

**填写后**：
```
尊敬的 张三：
您好！我们诚挚地邀请您出席 **2026黑客松大赛**。
活动时间：2026年5月1日 09:00 至 2026年5月3日 18:00
活动地点：南京市XX区XX路XX号
```

### 示例2：填写预算表

**原文**：
```
场地租赁：[预算金额]
餐饮费用：[预算金额]
物料制作：[预算金额]
```

**填写后**：
```
场地租赁：50000元
餐饮费用：30000元
物料制作：20000元
```

## 注意事项

1. **统一填写**：确保同一变量在所有文档中填写一致
2. **及时更新**：变量值变更时，同步更新所有相关文档
3. **备份原值**：填写前备份原始文档，便于后续修改
4. **团队同步**：变量值变更后，通知所有团队成员
