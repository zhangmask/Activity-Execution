# 07 物料设计与采购

> 本手册适用于活动所有物料的设计、采购、管理和回收。
> 参考案例：活动主办方 历史活动

---

## 一、核心原则

1. **物料清单要早定**：活动方案确定后立即梳理物料清单，避免临时加急
2. **设计稿统一管理**：所有设计稿集中存放，版本清晰
3. **提前下单**：印刷品至少提前7天，定制品至少提前14天
4. **专人管理物料**：采购、入库、发放、回收全流程有记录

## 三、设计流程

### 3.1 设计需求收集

1. 梳理所有需要设计的物料，建立**设计需求清单**
2. 明确每项物料的尺寸、数量、用途、截止时间
3. 确定主视觉风格（科技感、年轻化等）
4. 收集需要包含的信息（logo、二维码、文字等）

### 3.2 设计执行

1. 主KV先行：先确定主视觉，再延展到各物料
2. 设计稿评审：核心团队确认后才下单制作
3. 印刷前校对：文字、颜色、尺寸逐一核对
4. 打样确认：大批量印刷前先打样确认

### 3.3 设计稿管理

- 所有设计稿统一存放在共享文件夹
- 命名规范：物料类型_版本号_日期
- 保留源文件（PSD/AI/Figma链接）
- 最终版标记为"FINAL"

## 四、采购流程

### 4.1 供应商选择

- 至少询价3家供应商
- 比较价格、质量、交期
- 确认供应商能按时交货
- 大批量订单要求提供样品

### 4.2 采购时间线

| 时间节点 | 任务 |
|---|---|
| T-20天 | 梳理物料清单、确定数量 |
| T-15天 | 设计稿完成、询价比价 |
| T-10天 | 下单制作 |
| T-5天 | 到货验收、数量核对 |
| T-3天 | 物料分类打包、运送至场地 |
| T-1天 | 现场布置 |
| T日 | 物料发放 |
| T+1天 | 物料回收、盘点 |

### 4.3 验收标准

- 数量是否正确
- 质量是否达标（颜色、材质、印刷清晰度）
- 尺寸是否符合要求
- 有无损坏或瑕疵

## 六、易遗漏事项

1. **物料备份**：关键物料（胸牌、证书、评分表）多准备10-20%的备用量
2. **运输包装**：易碎物料需要专门包装，避免运输损坏
3. **物料存放**：提前确认存放空间，避免现场找不到
4. **发放记录**：物料发放要有记录，避免遗失
5. **回收计划**：可重复使用的物料（展架、横幅）需要回收计划
6. **现场补印**：准备1-2家能当天取货的快印店联系方式
7. **手持道具**：拍照打卡用的KT板等道具容易被忽略
8. **房间指示牌**：不同功能区域的指示牌容易遗漏

## 八、历史教训

> **历史活动 物料管理**：使用飞书多维表格管理物料清单，包含预算表中的物料清单和独立的设计需求清单。灵感墙文档用于UX与定制物料的设计参考。

> **历史活动 的教训**：物料设计需求清单应尽早定稿，设计稿统一汇总管理。手持KT版等互动道具容易被忽略，需要在物料清单中专门列出。
