# 5_2 如何规划展位布局

> 适用场景：已确定展商名单，需要规划展位的位置、动线和配套设施，确保展览区美观、有序、安全。
> 前置条件：[5_1_如何招募和管理展商](5_1_如何招募和管理展商.md)

---

## 第一步：了解展位类型

### 展位类型定义

| 展位类型 | 尺寸 | 标准配置 | 适合对象 |
|---|---|---|---|
| **标准展位** | 2m × 2m | 1桌2椅、1个电源、展位牌 | 普通展商 |
| **豪华展位** | 3m × 3m | 2桌4椅、2个电源、展位牌、展架 | 重点展商 |
| **赞助商展位** | 4m × 3m | 定制配置、醒目位置、灯箱 | 赞助商 |
| **互动体验区** | 3m × 4m | 更大空间、电源充足 | 需要互动体验的展品 |

### 展位配置清单

| 配置项 | 标准展位 | 豪华展位 | 赞助商展位 |
|---|---|---|---|
| 桌子 | 1张 | 2张 | 按需 |
| 椅子 | 2把 | 4把 | 按需 |
| 电源插座 | 1个 | 2个 | 3-4个 |
| 展位牌 | ✅ | ✅ | ✅（加大版） |
| 展架/展板 | ❌ | 1个 | 2个 |
| 灯光补充 | ❌ | ❌ | ✅ |
| WiFi | 共享 | 共享 | 专属 |

## 第二步：展位布局原则

### 核心原则

| 原则 | 说明 |
|---|---|
| **人流动线清晰** | 观众能自然地逛完所有展位，不走回头路 |
| **主次分明** | 赞助商/重点展商放在入口或主通道旁 |
| **安全第一** | 消防通道畅通，用电安全，不堵塞出口 |
| **美观统一** | 展位风格统一，标识清晰 |
| **动静分区** | 有互动体验的展位和安静展示的展位分开 |

### 展位位置价值评估

| 位置 | 人流程度 | 适合安排 | 价值等级 |
|---|---|---|---|
| 入口处 | 最高 | 赞助商展位、互动体验区 | ⭐⭐⭐⭐⭐ |
| 主通道两侧 | 高 | 豪华展位、重点展商 | ⭐⭐⭐⭐ |
| 中心区域 | 中高 | 标准展位 | ⭐⭐⭐ |
| 靠墙区域 | 中 | 标准展位 | ⭐⭐ |
| 角落区域 | 较低 | 备用展位、物料存放 | ⭐ |

## 第三步：设计动线

### 动线设计方法

**推荐动线类型：**

#### 1. 单向环形动线（推荐）

```
入口 → [展位1] → [展位2] → [展位3] → [展位4]
                                              ↓
出口 ← [展位8] ← [展位7] ← [展位6] ← [展位5]
```

优点：观众自然逛完所有展位，不遗漏，不拥堵。

#### 2. 主通道+支路动线

```
入口
  ↓
[主通道]
  ↓              ↓              ↓
[展位1-2]    [展位3-4]    [展位5-6]
  ↓              ↓              ↓
[主通道汇聚]
  ↓
出口
```

优点：适合展位数量较多的场景。

### 动线设计注意事项

- [ ] 主通道宽度至少 **2.5米**（容纳2人并排+1人逆行）
- [ ] 次通道宽度至少 **1.5米**
- [ ] 消防通道宽度至少 **1.2米**，不得被展位占用
- [ ] 入口和出口不要设在同一位置（避免人流对冲）
- [ ] 在动线交汇处设置指引牌
- [ ] 休息区/饮水区设在动线中段（让观众有休息点）

## 第五步：用电方案

### 用电需求统计

| 展位编号 | 展商名称 | 设备清单 | 功率（W） | 插座数量 |
|---|---|---|---|---|
| A1 | | 笔记本+显示器 | 300 | 2个 |
| A2 | | 笔记本+投影 | 400 | 2个 |
| B1 | | 笔记本 | 100 | 1个 |
| ... | | | | |
| **合计** | | | **W** | **个** |

### 用电安全规范

| 规范 | 要求 |
|---|---|
| 单个展位功率限制 | 不超过 500W（普通展位）/ 1000W（大展位） |
| 接线方式 | 使用带漏电保护的接线板，禁止私拉电线 |
| 线路管理 | 电线走线沿墙或用地线槽覆盖，不得裸露在通道上 |
| 大功率设备 | 如需使用加热设备、大屏幕等，需提前报备 |
| 应急措施 | 配电箱旁备有灭火器，技术组全程值守 |

### 用电方案 Checklist

- [ ] 统计所有展商用电需求
- [ ] 确认场地总用电负荷
- [ ] 每个展位分配插座和接线板
- [ ] 电线走线方案确定（沿墙/地线槽）
- [ ] 漏电保护器安装确认
- [ ] 配电箱位置确认
- [ ] 大功率设备单独回路
- [ ] 技术组人员值守安排

## 常见问题

**Q：展商想换展位怎么办？**
A：在展位分配时尽量公平（按报名顺序或抽签），提前在参展协议中说明"展位一经分配不可更换"。如有特殊情况（如两个展商协商互换），需经管理组同意。

**Q：展位不够用怎么办？**
A：①压缩通道宽度（但不能低于1.5米）；②将部分标准展位升级为紧凑型；③减少非必要功能区面积；④如果场地确实不够，考虑增加展区面积或减少展商数量。

**Q：展区太冷清怎么办？**
A：①将互动体验区放在展区入口，吸引人流；②在展区设置打卡点或小游戏；③安排主持人定时在展区做导览；④在展区设置游客投票点。

**Q：用电跳闸怎么办？**
A：提前了解场地总负荷，合理分配各展位功率。技术组备好电工工具和备用保险丝。跳闸时先断开大功率设备，再逐个合闸排查。

**Q：展区太热/太冷怎么办？**
A：提前确认场地空调/通风情况。如果场地没有空调，准备移动风扇或取暖设备。展位间距适当加大，保持空气流通。

## 历史教训

> 历史活动的经验：展位布局一定要提前在场地踩点时确认电源位置和数量，这是最容易被忽略但影响最大的因素。动线设计要让观众自然地从入口逛到出口，不走回头路。赞助商展位必须安排在入口或主通道旁最显眼的位置。展位牌建议统一设计风格，提前3天制作完成。用电方面，技术组必须全程值守，提前检查每个展位的电源和接线板。布局图确定后发给所有展商，让他们提前了解自己的位置和周边环境。
