# 5_1 如何招募和管理展商

> 适用场景：活动中设有展览/展示区，需要招募参展方（展商）并管理其参展全过程。
> 前置条件：[3_3_如何布置和验收场地](3_3_如何布置和验收场地.md)

---

## 第一步：明确展商定位和需求

### 确定你需要什么样的展商

| 问题 | 你的答案 |
|---|---|
| 展览主题是什么？ | 如：AI产品、创业项目、科技硬件 |
| 需要多少个展位？ | 个 |
| 展商需要付费吗？ | 是/否，价格多少？ |
| 展商面向谁展示？ | 选手/游客/投资人/媒体 |
| 展期多长？ | 天 |

### 展商筛选标准

| 标准 | 必须满足 | 加分项 |
|---|---|---|
| 与活动主题相关 | ✅ | — |
| 产品/项目可展示（有实物或Demo） | ✅ | 有互动体验 |
| 有基本的展示能力（海报、物料） | ✅ | 自带展示设备 |
| 有专人负责现场 | ✅ | 团队多人到场 |
| 品牌形象正面 | ✅ | 知名度高 |
| 能提供赞助或资源支持 | — | ✅ |

## 第二步：制定招募时间线

| 时间节点 | 任务 | 负责人 |
|---|---|---|
| T-6周 | 确定展位数量、类型、价格 | 策划组 |
| T-5周 | 制作招募海报和参展方指南 | 设计组 |
| T-5周 | 启动招募（发布招募信息） | 宣发组 |
| T-4周 | 第一轮跟进，确认意向展商 | 商务组 |
| T-3周 | 截止报名，确认最终展商名单 | 商务组 |
| T-2周 | 收集展商信息（展品、用电、布展需求） | 商务组 |
| T-1周 | 发送参展方指南、确认布展时间 | 商务组 |
| T-1天 | 展商布展 | 场务组配合 |
| T日 | 活动当天展商管理 | 现场管理组 |
| T+1天 | 撤展 | 场务组配合 |

## 第三步：招募展商

### 招募渠道

| 渠道 | 适合找什么展商 | 操作方式 |
|---|---|---|
| 合作方推荐 | 相关企业/项目 | 请赞助商、合作方推荐 |
| 社交媒体 | 创业团队、独立开发者 | 在小红书、微信公众号发布招募帖 |
| 创业社群 | 创业项目、初创公司 | 在相关社群转发 |
| 高校社团 | 学生项目、科研成果 | 联系高校创业中心、科技社团 |
| 往届参与者 | 有展示需求的老朋友 | 直接联系 |

## 第六步：展位类型和价格

### 展位类型设计

| 展位类型 | 尺寸 | 配置 | 建议价格 |
|---|---|---|---|
| **标准展位** | 2m × 2m | 1张桌子 + 2把椅子 + 1个电源 + 展位牌 | 500-1000元 |
| **豪华展位** | 3m × 3m | 2张桌子 + 4把椅子 + 2个电源 + 展位牌 + 展架 | 1000-2000元 |
| **赞助商展位** | 4m × 3m | 定制配置 + 醒目位置 + 额外宣传 | 含在赞助权益中 |

> 注：根据活动性质，展商可能是免费参展（赞助商权益的一部分）或付费参展。

## 第七步：布展撤展安排

### 布展安排

| 时间 | 任务 | 注意事项 |
|---|---|---|
| T-1天 下午 | 展商入场布展 | 安排工作人员引导，提供手推车 |
| T-1天 下午 | 检查各展位用电 | 技术组逐个检查电源和设备 |
| T-1天 晚上 | 布展完成、最终检查 | 确认所有展位就位、安全检查 |

### 布展 Checklist

- [ ] 展位桌椅到位
- [ ] 展位牌安装
- [ ] 电源接通并测试
- [ ] 展商物料到位
- [ ] 展商设备调试完成
- [ ] 通道宽度确认（至少1.2米）
- [ ] 消防通道确认畅通

### 撤展安排

| 时间 | 任务 | 注意事项 |
|---|---|---|
| 活动结束日 | 展商撤展 | 安排工作人员协助 |
| 撤展后 | 场地清理 | 检查有无遗留物品 |
| 撤展后 | 设备归还 | 归还借用的设备 |

### 撤展 Checklist

- [ ] 展商物品全部撤走
- [ ] 借用设备归还
- [ ] 场地清理干净
- [ ] 检查有无损坏物品
- [ ] 遗留物品拍照登记

## 常见问题

**Q：招不到展商怎么办？**
A：①降低或免除展位费，以赞助权益置换；②扩大招募渠道，多在创业社群宣传；③邀请合作方推荐；④如果实在招不满，可以将展区面积缩小，集中展示。

**Q：展商临时取消怎么办？**
A：提前在参展协议中写明取消条款（如提前X天通知可全额退款）。准备2-3个候补展商。如果展商临时不来，可将空展位改为互动区或展示区。

**Q：展商展品太多放不下怎么办？**
A：提前在参展方指南中明确展位面积和物品限制。如果展品确实很多，可以协商升级展位或加购展位。

**Q：展商之间发生纠纷怎么办？**
A：在参展协议中明确现场规则（噪音、占道、宣传等），安排现场管理人员及时协调。

**Q：展商用电超负荷怎么办？**
A：提前了解场地电路负荷，对每个展位限制用电功率（如不超过500W）。大功率设备（如加热设备）需提前报备。

## 历史教训

> 历史活动的经验：展商招募要尽早启动（至少提前5周），并制作专业的参展方指南发给每个展商。展商信息收集务必在活动前2周完成，特别是用电需求和设备需求，便于现场提前安排。布展当天安排专人对接每个展商，提供手推车等搬运工具。展位牌提前制作，包含展商名称和展位编号。撤展时要提醒展商带走所有物品，避免遗留。
