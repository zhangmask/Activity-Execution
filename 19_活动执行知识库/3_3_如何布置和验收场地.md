# 3_3 如何布置和验收场地

> 适用场景：活动前一天或当天，需要布置场地并验收。
> 前置条件：[3_2_如何做场地功能分区](3_2_如何做场地功能分区.md)

---

## 第一步：制定布置计划

### 布置时间线

| 时间 | 任务 | 负责人 |
|---|---|---|
| T-3天 | 大件物料进场（帐篷、展架、桌椅） | 场务组 |
| T-2天 | 舞台搭建、设备安装 | 场务组+技术组 |
| T-1天 | 全面布置（装饰、标识、物料摆放） | 全员 |
| T-1天下午 | 设备调试（音响、灯光、投影） | 技术组 |
| T-1天晚上 | 最终检查、拍照记录 | 总协调 |

### 人员分工

| 组别 | 负责区域 | 人数 |
|---|---|---|
| 签到组 | 签到区布置 | 2人 |
| 主会场组 | 舞台、座椅、设备 | 4-6人 |
| 展区组 | 展位、标识、装饰 | 3-4人 |
| 后勤组 | 物料搬运、后勤区 | 2-3人 |
| 技术组 | 所有设备调试 | 2-3人 |

---

## 第二步：布置执行

### 签到区布置

- [ ] 签到桌摆放（2-3张桌子）
- [ ] 签到表/签到二维码打印
- [ ] 物料包准备（胸牌、手册、手环）
- [ ] 指引牌摆放
- [ ] 签到区装饰

### 主会场布置

- [ ] 舞台搭建/确认
- [ ] LED屏/投影安装和测试
- [ ] 音响系统安装和测试
- [ ] 灯光安装和测试
- [ ] 座椅摆放（按座位图）
- [ ] 桌签摆放
- [ ] 水/饮品摆放
- [ ] 摄影机位确认

### 展区布置

- [ ] 展位桌椅摆放
- [ ] 展位牌安装
- [ ] 电源接线
- [ ] 展区标识
- [ ] 主通道保持畅通

### 其他区域

- [ ] Workshop区桌椅和设备
- [ ] 休息区桌椅和饮品
- [ ] 嘉宾休息室布置
- [ ] 后勤区物料整理
- [ ] 所有区域的指引标识

---

## 第三步：设备调试

### 调试清单

| 设备 | 调试内容 | 通过标准 |
|---|---|---|
| LED屏/投影 | 分辨率、色彩、亮度 | 清晰可见，无坏点 |
| 音响 | 音量、音质、话筒 | 全场能听清，无杂音 |
| 灯光 | 各场景切换 | 演讲/休息/活动模式正常 |
| 网络 | WiFi连接数、速度 | 能同时连接目标人数 |
| 电脑 | PPT切换、视频播放 | 流畅无卡顿 |
| 直播 | 推流、画面、声音 | 画面清晰，声音同步 |
| 空调 | 温度、风量 | 温度适宜 |

### 测试方法

1. **音量测试**：在场地最远处能否听清
2. **网络压力测试**：让10-20人同时连接WiFi
3. **PPT测试**：测试所有格式的PPT（16:9、4:3）
4. **视频测试**：播放视频检查声音和画面同步
5. **备用方案测试**：测试备用电脑、备用话筒

---

## 第四步：最终验收

布置完成后，总协调人进行最终验收：

### 验收 Checklist

#### 外观
- [ ] 所有区域整洁干净
- [ ] 装饰物固定牢靠
- [ ] 标识清晰可见
- [ ] 无安全隐患（电线裸露、地面湿滑等）

#### 设备
- [ ] 所有设备正常运行
- [ ] 备用设备就位
- [ ] 充电设备充足

#### 物料
- [ ] 所有物料到位
- [ ] 物料数量核对
- [ ] 物料分类存放

#### 动线
- [ ] 各区域动线清晰
- [ ] 紧急出口畅通
- [ ] 指引标识完整

#### 安全
- [ ] 消防通道畅通
- [ ] 灭火器就位
- [ ] 急救箱就位
- [ ] 无安全隐患

---

## 第五步：拍照记录

验收通过后，对所有区域拍照记录：

- 每个区域的全景照
- 关键设备的特写照
- 物料摆放的照片
- 舞台/主会场的照片

这些照片用于：
- 活动后的复盘
- 下次活动的参考
- 宣发素材

## 常见问题

**Q：物料到不了怎么办？**
A：提前联系快印店，准备当天取货的备选方案。关键物料（胸牌、证书）多准备备用量。

**Q：设备调试有问题怎么办？**
A：联系场地技术人员或设备供应商。如果无法解决，启用备用设备。

**Q：布置时间不够怎么办？**
A：增加人手，优先布置核心区域（主会场、签到区），非核心区域可以简化。

## 历史教训

> 历史活动的经验：所有电子文件（PPT、视频、BGM、主持词、流程表、座位图）必须提前收齐。彩排非常重要，PPT格式比例、音频、视频、话筒、摄影摄像机位都需要提前确认。
