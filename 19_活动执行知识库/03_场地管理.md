# 03 场地管理

> 本手册适用于活动场地的选择、确认、布置和管理全流程。
> 参考案例：活动主办方 历史活动（场地A + 场地B双会场）

---

## 一、核心原则

1. **场地越早确认越好**：场地是所有后续工作的基础，场地不定则物料、动线、设备全部悬空
2. **亲自踩点**：不能只看照片和参数，必须到场实地测量
3. **备用方案**：至少准备一个备选场地，应对突发变更
4. **费用全口径**：场地费用不只是租金，还包括设备、安保、网络、清洁、停车等

---

## 二、场地选择 Checklist

### 2.1 基础条件

- [ ] 容纳人数（区分主会场、分会场、休息区、展区）
- [ ] 交通便利性（地铁/公交/停车场）
- [ ] 无障碍设施
- [ ] 网络条件（WiFi承载人数、有线网络）
- [ ] 电力容量（尤其是展区用电、大屏设备）
- [ ] 空调/供暖（五一/十一期间可能不供暖）

### 2.2 设备条件

- [ ] LED屏/投影（尺寸、分辨率、接口）
- [ ] 音响系统（话筒数量、功放功率）
- [ ] 灯光（舞台灯光、环境灯光）
- [ ] 讲台/舞台
- [ ] 控台位置

### 2.3 服务配套

- [ ] 餐饮供应能力（食堂/外卖/茶歇）
- [ ] 住宿条件（是否有配套酒店/宿舍）
- [ ] 安保服务
- [ ] 清洁服务
- [ ] 停车场（车位数量、停车券）

## 三、场地规划流程

### 3.1 功能分区

按活动需求划分区域：

| 区域 | 功能 | 需要面积参考 |
|---|---|---|
| 主会场 | 开幕式、路演、颁奖 | 按人数×1.5㎡ |
| 开发区 | 选手通宵开发 | 每组2-3㎡ |
| 展区 | 展商展位、游园会 | 每展位6-9㎡ |
| Workshop区 | 技术工作坊 | 50-100人容量 |
| 休息区 | 选手休息、社交 | 按人数×1㎡ |
| 后勤区 | 物料存放、工作人员休息 | 15-30㎡ |
| 签到区 | 入口签到、咨询 | 10-20㎡ |

### 3.2 动线设计

- **选手动线**：签到 → 开发区 → 展区 → 主会场
- **游客动线**：签到 → 展区 → 主会场（需要门禁隔离）
- **嘉宾动线**：VIP通道 → 休息室 → 主会场 → 展区
- **工作人员动线**：后勤通道，不与选手/游客交叉

### 3.3 用电方案

- 列出每个区域的用电设备和功率
- 计算总功率，确认场地供电容量
- 准备足够的排插和拖线盘
- 标注取电点位，方便布线

## 四、场地布置执行

### 4.1 布置时间线

| 时间节点 | 任务 | 负责人 |
|---|---|---|
| T-7天 | 确认所有物料到场时间 | 物料组 |
| T-3天 | 大件物料进场（帐篷、展架） | 场务组 |
| T-1天 | 全面布置（桌椅、装饰、设备调试） | 全员 |
| T日 | 最终检查、细节调整 | 总协调 |

### 4.2 设备调试清单

- [ ] 所有屏幕显示正常（分辨率、色彩）
- [ ] 音响测试（话筒、功放、降噪）
- [ ] 网络测速（同时连接人数测试）
- [ ] 灯光场景切换（演讲模式、休息模式、活动模式）
- [ ] PPT切换测试（备用电脑）
- [ ] 直播设备测试（推流、画面、声音）
- [ ] 空调/通风测试

## 五、多场地协同（如适用）

如果活动在多个场地举行，需要特别注意：

1. **交通接驳**：安排大巴/摆渡车，明确发车时间和路线
2. **时间衔接**：预留充足的转场时间，避免环节延误
3. **信息同步**：每个场地设一个现场负责人，保持通讯畅通
4. **物料分配**：提前规划哪些物料在哪个场地，避免搬运遗漏
5. **门禁管理**：不同场地可能有不同的门禁权限，提前申请

---

## 六、易遗漏事项

1. **场地合同**：口头确认不可靠，必须签书面合同，明确费用、设备、时间、违约责任
2. **场地对接人**：确认场地方面的对接人和联系方式，了解场地内部流程
3. **停车场券**：如果需要免费停车，提前向场地申请停车券
4. **货梯和进场通道**：大型物料需要确认货梯尺寸、进场时间限制
5. **持续供电供冷**：五一/十一假期期间，场地可能不持续供电供冷，需要特别确认
6. **消防审批**：超过一定人数需要消防报备
7. **垃圾清运**：活动后的垃圾清运责任和费用
8. **桌椅借用**：确认场地提供多少桌椅，不够的部分需要额外租赁
9. **雨天预案**：室外场地需要准备雨棚或备用室内方案

## 八、历史教训

> **历史活动 场地**：采用场地A（主开发区+展区）+ 场地B（开幕式+游园会+路演）的双会场模式。需要大巴接驳，增加了大量协调成本。

> **历史活动 的教训**：场地确认应尽早，场地不定则后续所有工作（物料设计、动线规划、设备方案）都无法推进。五一期间场地A的持续供电供冷需要特别确认。
