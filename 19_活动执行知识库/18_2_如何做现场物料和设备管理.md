> **适用场景**：黑客松活动现场的物料核对、设备管理、后勤保障。
> **前置条件**：请先完成 `18_1_如何做现场调度和应急预案.md`，确认现场指挥体系。

## 第一步：物料清单核对方法

### 物料分类管理

| 类别 | 物料类型 | 管理方式 |
|------|---------|---------|
| 宣传物料 | 海报、展架、横幅、背景板 | 提前制作，活动前1天进场 |
| 签到物料 | 手环、签到表、资料袋、桌签 | 提前分装，按颜色分类 |
| 技术设备 | 投影仪、音响、电脑、摄像头 | 编号管理，借用登记 |
| 办公用品 | 笔、纸、便签、胶带、剪刀 | 按区域分配 |
| 餐饮水具 | 瓶装水、纸杯、咖啡、茶包 | 按人数+20%备用 |
| 装饰物料 | 气球、彩带、花艺、指示牌 | 活动前布置 |
| 急救物资 | 急救箱、AED、防暑药品 | 固定位置，专人管理 |
| 其他 | 垃圾袋、清洁用品、备用物资 | 充足备用 |

### 核对流程

```
第一步：制作物料清单
  - 各组提交物料需求
  - 物资组汇总整理
  - 总指挥审批

第二步：采购和制作
  - 确认供应商和到货时间
  - 预留制作和运输时间
  - 到货后逐一清点

第三步：进场前核对
  - 按清单逐项核对数量
  - 检查物料质量（破损、色差等）
  - 标记已核对

第四步：分区域分配
  - 按区域需求分配物料
  - 每个区域指定物料负责人
  - 制作区域物料清单

第五步：活动后回收
  - 各区域回收剩余物料
  - 核对消耗情况
  - 可重复使用的物料标记保存
```

### 物料发放原则

```
1. 按需发放：根据实际需求发放，不超量
2. 签字确认：每次发放必须签字
3. 专人管理：每个区域指定物料负责人
4. 定期盘点：每半天盘点一次消耗情况
5. 及时补充：消耗超过70%时申请补充
```

## 第三步：设备管理

### 设备编号管理

#### 设备编号规则

```
编号格式：{设备类型}-{编号}

设备类型代码：
- PJ = 投影仪 (Projector)
- SP = 音箱 (Speaker)
- MC = 麦克风 (Microphone)
- PC = 电脑 (Computer)
- CM = 摄像头 (Camera)
- NT = 网络设备 (Network)
- PS = 电源/插排 (Power Strip)
- OT = 其他 (Other)

示例：PJ-001（1号投影仪）、SP-002（2号音箱）
```

### 设备管理流程

```
借出流程：
  1. 借用人填写借用登记表
  2. 物资组检查设备状态
  3. 双方确认设备外观和功能
  4. 签字借出

归还流程：
  1. 借用人归还设备
  2. 物资组检查设备状态
  3. 确认无损坏后签字归还
  4. 如有损坏，记录并报修

故障处理流程：
  1. 发现设备故障
  2. 立即报告物资组
  3. 物资组评估是否可现场修复
  4. 可修复：安排技术人员维修
  5. 不可修复：启用备用设备
  6. 记录故障信息
```

## 第四步：工作人员餐饮安排

### 餐饮安排方案

| 餐次 | 时间 | 人数 | 餐饮类型 | 预算 | 供应商 |
|------|------|------|---------|------|--------|
| 早餐 | 07:30-08:30 | 工作人员 | 面包+牛奶+咖啡 | 15元/人 | |
| 午餐 | 12:00-13:00 | 全员 | 盒饭（荤素搭配） | 25-35元/人 | |
| 晚餐 | 18:00-19:00 | 全员 | 盒饭/自助 | 25-35元/人 | |
| 宵夜 | 22:00-23:00 | 选手+工作人员 | 面包+泡面+水果 | 15元/人 | |
| 茶歇 | 全天 | 全员 | 咖啡+茶+零食 | 10元/人 | |

### 餐饮管理要点

```
1. 提前确认饮食禁忌
   - 报名表中收集饮食禁忌信息
   - 汇总后告知餐饮供应商
   - 准备素食/清真/无麸质等特殊餐食

2. 分时段供餐
   - 避免所有人同时取餐造成排队
   - 建议分3批：工作人员→选手→嘉宾

3. 食品安全
   - 确认供应商食品经营许可证
   - 餐食送达后检查温度和包装
   - 保留样品备查（冷藏保存48小时）

4. 餐饮区管理
   - 设置明显的餐饮区标识
   - 配备垃圾桶和清洁用品
   - 安排志愿者维持秩序和清洁
```

### 餐饮检查清单

- [ ] 餐饮供应商确认

## 第五步：充电设备管理

### 充电需求分析

| 设备 | 充电需求 | 解决方案 |
|------|---------|---------|
| 手机 | 高（每人至少1次/天） | 设置充电区 |
| 笔记本电脑 | 高（选手持续使用） | 确保座位有插座 |
| 平板 | 中 | 充电区 |
| 对讲机 | 低（活动前后） | 后台集中充电 |
| 相机/摄像机 | 中 | 后台充电 |
| 无线麦克风 | 低 | 后台充电 |

### 充电区设置方案

```
位置：设置在休息区或走廊显眼位置
配置：
  - 多口充电排插：至少5个（每排插8-10个插孔）
  - USB充电线：准备Type-C、Lightning、Micro-USB各10根
  - 充电宝：准备10-20个（可供临时借用）
  - 标识牌："充电区"醒目标识

管理：
  - 安排1人看管充电区
  - 充电宝借用需登记（手机尾号+姓名）
  - 提醒选手不要长时间占用插座
  - 活动结束前30分钟提醒取回充电设备
```

### 电源保障方案

| 项目 | 数量 | 位置 | 用途 |
|------|------|------|------|
| 8孔插排 | 20个 | 各座位区 | 选手笔记本供电 |
| 4孔插排 | 10个 | 充电区 | 手机充电 |
| 大功率插排 | 5个 | 技术台 | 设备供电 |
| UPS不间断电源 | 2台 | 技术台 | 关键设备应急供电 |
| 延长线 | 10根 | 各区域 | 扩展电源覆盖 |

### 电源检查清单

- [ ] 确认场地总电源容量
- [ ] 各区域电源点位确认
- [ ] 插排数量充足
- [ ] 充电区布置完成
- [ ] 充电线类型齐全
- [ ] 充电宝充满电
- [ ] UPS测试正常

### 清洁管理方案

```
1. 清洁人员配置
   - 安排2-3名清洁志愿者（或聘请保洁）
   - 分区域负责

2. 清洁时间表
   - 每2小时巡查清理一次垃圾桶
   - 每餐后立即清理餐饮区
   - 活动结束后全面清扫

   - 垃圾袋（大/中/小各100个）
   - 扫帚和簸箕（3套）
   - 拖把和水桶（2套）
   - 清洁湿巾（5包）
   - 一次性手套（2盒）

4. 场地归还标准
   - 所有垃圾清理完毕
   - 桌椅归位
   - 地面清洁
   - 设施完好
   - 与物业验收确认
```

## 常见问题

**Q1：物料到了发现数量不对怎么办？**
A：①立即联系供应商补货/退换；②如果来不及补货，优先保障核心物料（手环、资料袋）；③非核心物料可以减少或替代；④在物料清单中记录差异。

**Q2：设备临时借用怎么办？**
A：①优先使用团队成员自有设备；②联系学校/公司IT部门借用；③联系设备租赁公司（提前建立联系）；④在社群中询问是否有成员可以提供。

**Q3：选手反映插座不够怎么办？**
A：①立即增加插排数量；②在座位区之间增设电源延长线；③引导部分选手到备用区域（通常有更多插座）；④提醒选手使用低功耗模式。

**Q4：餐饮送晚了怎么办？**
A：①提前1小时联系供应商确认送餐时间；②准备应急零食（面包、饼干）；③如果延迟超过30分钟，安排志愿者分发零食安抚；④调整议程，将休息时间后移。

**Q5：活动结束后场地清洁不合格怎么办？**
A：①活动结束前1小时开始初步清理；②安排足够人手（至少5人）参与清理；③按照场地归还标准逐一检查；④不合格的项目立即整改；③与物业一起验收确认。

## 历史教训

- 历史活动物料管理的核心是"提前准备、按需分配、及时盘点"
- 设备编号管理是避免混乱的关键，每台设备都要贴标签
- 餐饮是选手满意度的重要影响因素，不要在餐饮上省钱
- 充电保障是黑客松的刚需，确保每个座位都有电源
- 历史教训表明：物料多准备20%的备用量是最安全的
- 垃圾处理看似小事，但直接影响场地归还和物业关系
