# 如何做现场调度和应急预案

> **适用场景**：黑客松活动现场的统一调度指挥、突发情况应急处理。
> **前置条件**：请先完成 `03_场地管理.md`，确认场地布局和设施情况。

## 第一步：建立现场指挥体系

### 现场指挥架构

```
                    ┌─────────────┐
                    │  总指挥      │
                    │ （总负责人）  │
                    └──────┬──────┘
                           │
            ┌──────────────┼──────────────┐
            │              │              │
     ┌──────┴──────┐ ┌────┴────┐  ┌──────┴──────┐

    ┌───────┼───────┐     │       ┌───────┼───────┐
    │       │       │     │       │       │       │
```

| 岗位 | 人数 | 核心职责 | 关键权限 |

| 现场执行官 | 1-2人 | 现场流程协调、时间控制 | 流程调整权 |
| 安全官 | 1人 | 安全巡查、应急处置 | 安全决策权、紧急叫停权 |
| 后勤保障官 | 1人 | 物资、餐饮、设备保障 | 后勤调配权 |

## 第二步：搭建通讯体系

### 通讯工具配置

| 工具 | 用途 | 使用场景 |
|------|------|---------|
| 对讲机 | 核心团队即时通讯 | 现场调度、紧急情况 |
| 微信群 | 全员沟通 | 信息发布、非紧急沟通 |
| 电话 | 紧急联络 | 对讲机失灵时的备用 |
| 哨子/警报 | 紧急信号 | 紧急疏散、重大事故 |

### 对讲机频道分配

| 频道 | 使用人员 | 用途 |
|------|---------|------|
| CH1 | 总指挥+各组组长 | 全局调度 |
| CH2 | 签到组+安保组 | 入口管理 |
| CH3 | 活动组+技术组 | 环节执行 |
| CH4 | 后勤组+医疗组 | 后勤保障 |

### 通讯规范

```
对讲机使用规范：
1. 呼叫格式："XX组呼叫总指挥，收到请回复"
2. 回复格式："总指挥收到，请讲"
3. 信息格式："时间+地点+事件+需要的支持"
4. 示例："10:30，主会场，投影仪故障，需要技术组支援"
5. 紧急情况：连呼三遍"紧急"，所有人保持静默

微信群使用规范：
1. 重要信息@相关人
2. 紧急信息发完后电话确认
3. 不发无关信息，保持频道清晰
4. 每30分钟各组汇报一次状态
```

## 第三步：现场执行流程

### T-1天（活动前一天）

| 时间 | 任务 | 负责人 | 检查项 |
|------|------|--------|--------|
| 09:00 | 场地验收 | 后勤保障官 | 场地设施完好 |
| 10:00 | 物资进场 | 物资组 | 清点核对 |
| 11:00 | 设备安装调试 | 技术组 | 网络/投影/音响测试 |
| 14:00 | 场地布置 | 全员 | 桌椅/标识/装饰 |
| 16:00 | 签到区布置 | 签到组 | 签到台/手环/资料 |
| 17:00 | 全流程彩排 | 现场执行官 | 各环节走一遍 |
| 18:00 | 问题整改 | 各组 | 修复彩排发现的问题 |
| 19:00 | 最终确认 | 总指挥 | 签字确认就绪 |

### T日（活动当天）

| 时间 | 任务 | 负责人 | 备注 |
|------|------|--------|------|
| T-3h | 工作人员到场 | 各组组长 | 签到、领取对讲机 |
| T-2h | 设备最终检查 | 技术组 | 网络/投影/音响/直播 |
| T-1.5h | 物资就位 | 物资组 | 各区域物资到位 |
| T-1h | 签到台开放 | 签到组 | 开始签到 |
| T-30min | 嘉宾/评委到场 | 活动组 | VIP接待 |
| T-15min | 最终确认 | 现场执行官 | 各组汇报就绪 |
| T-0 | 活动开始 | 总指挥 | 开幕式 |
| 进行中 | 各环节执行 | 活动组 | 按议程推进 |
| 进行中 | 现场巡查 | 安全官 | 每30分钟巡查一次 |
| 结束 | 清场收尾 | 全员 | 物资回收、场地清理 |

## 第四步：签到管理

### 签到流程

```
签到台配置（建议4-5个台）：
  1号台：VIP/嘉宾（快速通道）
  2号台：选手（A-M姓氏）
  3号台：选手（N-Z姓氏）
  4号台：观众/媒体
  5号台：现场报名/问题处理

签到流程：
  1. 游客出示报名凭证（电子票/短信）
  2. 工作人员核验身份信息
  3. 发放对应颜色手环
  4. 发放资料袋（含活动手册、地图、小礼品）
  5. 引导至对应区域
```

### 签到异常处理

| 异常情况 | 处理方式 |
|---------|---------|
| 查不到报名信息 | 让游客提供手机号/姓名，搜索后台；仍找不到则引导至5号台现场报名 |
| 报名信息与本人不符 | 要求出示身份证核实；无法核实则拒绝入场 |
| 手环颜色发完 | 立即联系物资组补充；临时使用其他颜色替代并记录 |
| 签到系统崩溃 | 启用纸质签到表备份；拍照记录签到者信息，事后补录 |

## 第五步：嘉宾接待

### 嘉宾接待流程

```
第一步：提前确认
  - 活动前3天：电话/微信确认出席
  - 活动前1天：发送提醒短信（含时间、地点、交通指引）
  - 活动当天：提前1小时再次确认

第二步：到场接待
  - 安排专人在停车场/地铁口迎接
  - 引导至嘉宾休息室
  - 提供茶水、点心
  - 递上活动流程表

第三步：活动期间
  - 专人陪同，随时响应需求
  - 提前5分钟提醒下一个环节
  - 确保嘉宾饮食、休息需求

第四步：离场送别
  - 感谢参与，赠送伴手礼
  - 安排车辆/引导至交通站点
  - 活动后发送感谢信
```

## 第六步：秩序维护

### 秩序维护要点

| 场景 | 风险 | 维护措施 |
|------|------|---------|
| 签到区 | 拥挤排队 | 设置排队隔离带，安排引导志愿者 |
| 主会场 | 座位不足 | 提前规划座位，爆满时引导至备用区 |
| 比赛区 | 非参赛人员闯入 | 手环验证，安排值守人员 |
| 展示区 | 设备损坏 | 设置"请勿触摸"标识，安排看护 |
| 出入口 | 人流交叉 | 单向流动设计，关键路口安排引导 |
| 餐饮区 | 排队过长 | 分时段供餐，设置多个取餐点 |

### 秩序维护志愿者配置

| 位置 | 人数 | 工作内容 |
|------|------|---------|
| 入口 | 2人 | 验证手环、引导入场 |
| 主会场 | 2人 | 维持秩序、引导座位 |
| 比赛区 | 2人 | 防止非授权进入 |
| 走廊/通道 | 2人 | 引导方向、防止聚集 |
| 出口 | 1人 | 引导离场 |

## 第八步：应急响应流程

### 一般应急响应流程

```
发现异常
    ↓
第一时间报告（对讲机/电话）
    ↓
相关负责人到场评估
    ↓
启动对应预案
    ↓
执行应急措施
    ↓
持续监控直到解决
    ↓
记录事件经过
    ↓
事后复盘总结
```

### 重大事件升级流程

```
事件发生
    ↓
现场人员初步处置（1分钟内）
    ↓
同时报告：安全官 + 总指挥
    ↓
安全官评估严重程度：
  - 一般事件：安全官指挥处置
  - 重大事件：总指挥接管指挥
  - 紧急事件：总指挥决定是否报警/疏散
    ↓
总指挥下达指令
    ↓
各组执行指令
    ↓
对外统一口径（仅总指挥可对外发言）
    ↓
事件结束后全体复盘
```

### 疏散流程

```
触发条件：
  - 火灾/地震
  - 安全威胁
  - 总指挥下达疏散指令

疏散步骤：
  1. 安全官通过对讲机下达："全体注意，启动疏散程序"
  2. 各区域负责人引导人员从最近的安全出口撤离
  3. 撤离顺序：伤病人员 → 儿童/老人 → 普通人员 → 工作人员
  4. 到达指定集合点后，各组清点人数
  5. 向总指挥汇报人数
  6. 确认全员安全后，等待进一步指令

疏散路线：
  - 每个区域至少2条疏散路线
  - 疏散路线标识醒目
  - 安排人员在关键节点引导
```

## 常见问题

**Q1：对讲机不够用怎么办？**
A：①优先分配给各组组长和关键岗位；②使用微信群作为补充通讯工具；③租赁额外对讲机（活动设备租赁公司通常提供）。

**Q2：嘉宾突然取消出席怎么办？**
A：①提前准备1-2位备选嘉宾；②调整议程，将其他环节提前；③安排团队成员临时补位做简短分享；④及时通知观众调整信息。

**Q3：现场有人晕倒怎么处理？**
A：①医疗组立即到场；②将患者移至通风处；③判断意识和呼吸；④轻微情况现场处理；⑤严重情况拨打120并通知家属；⑥记录事件经过。

**Q4：活动严重超时怎么办？**
A：①现场执行官评估剩余环节；②与总指挥商量是否压缩或删除某些环节；③通知主持人调整节奏；④提前告知观众预计延迟时间。

**Q5：下雨天签到区在户外怎么办？**
A：①提前准备雨棚/帐篷；②准备一次性雨衣给排队人员；③签到台转移至室内（如有条件）；④安排志愿者为排队人员撑伞。

## 历史教训

- 历史活动现场调度的核心是"一个指挥、一个声音"，避免多头指挥
- 对讲机是现场最重要的通讯工具，建议核心岗位人手一台
- 历史教训表明：80%的突发情况可以通过提前彩排发现和预防
- 疏散演练建议在T-1天进行，让所有工作人员熟悉疏散路线
- 应急联系表要打印多份，贴在签到台、后台工作区、急救点等关键位置
- 历史活动通常设置"现场指挥部"（一个固定位置），所有对讲机通讯在此汇总
