# 如何邀请嘉宾

> **适用场景**：hackathon活动需要邀请行业嘉宾参与开幕演讲、技术分享、评审指导等环节。
> **前置条件**：已完成活动整体规划和嘉宾需求分析（参考 2_2_嘉宾需求分析）。

---

## 第一步：确定嘉宾分类

### 嘉宾类型和角色

| 嘉宾类型 | 角色 | 邀请难度 | 建议人数 |
|----------|------|----------|----------|
| **开幕嘉宾** | 开幕式致辞、主题演讲 | ⭐⭐⭐⭐⭐ | 1-2人 |
| **导师/技术嘉宾** | 技术分享、现场指导 | ⭐⭐⭐ | 5-10人 |
| **评审嘉宾** | 项目评审、路演点评 | ⭐⭐⭐ | 3-5人 |
| **赞助商代表** | 赞助商致辞、颁奖 | ⭐⭐ | 按赞助协议 |
| **媒体嘉宾** | 活动报道、宣传 | ⭐⭐ | 2-3人 |

### 嘉宾选择标准

| 维度 | 评估要点 |
|------|----------|
| **行业影响力** | 在相关领域有知名度和影响力 |
| **内容质量** | 能带来有价值的分享和指导 |
| **受众匹配** | 与选手群体（学生/职场人）匹配 |
| **时间可用** | 活动时间是否方便 |
| **过往经历** | 是否有hackathon/技术活动经验 |
| **成本预算** | 差旅、出场费是否在预算内 |

---

## 第二步：做好邀请准备

### 邀请前准备清单

- [ ] 确定嘉宾名单（正式名单+备选名单）
- [ ] 收集嘉宾信息（职位、研究方向、社交媒体）
- [ ] 准备活动介绍材料（活动简介、往届回顾、日程安排）
- [ ] 确定邀请方式（邮件/微信/电话/当面）
- [ ] 确定对接人（谁负责联系哪位嘉宾）
- [ ] 准备嘉宾权益说明（差旅报销、出场费、礼品等）

### 邀请优先级

1. **先邀请确定性高的嘉宾**：往届合作过的、关系好的
2. **再邀请行业大咖**：影响力大但不确定的
3. **最后邀请备选嘉宾**：作为替补

## 第四步：跟进邀请节奏

### 邀请跟进时间线

| 时间 | 动作 | 说明 |
|------|------|------|
| 活动前6-8周 | 发送首次邀请 | 给嘉宾充足的时间考虑 |
| 活动前5周 | 第一次跟进 | 如未回复，礼貌跟进 |
| 活动前4周 | 确认最终名单 | 明确嘉宾是否参加 |
| 活动前3周 | 收集嘉宾信息 | 收集演讲主题、PPT、个人简介等 |
| 活动前2周 | 确认细节 | 确认行程、接待安排 |
| 活动前1周 | 最终确认 | 再次确认出席，发送日程 |

### 跟进话术

**第一次跟进（邀请发出后1周）**：
```
[嘉宾姓名]老师，您好！

上周给您发了一封关于[活动名称]嘉宾邀请的邮件，不知道您是否有机会看到？

如果您需要更多活动信息，我很乐意提供。期待您的回复！
```

**第二次跟进（邀请发出后2周）**：
```
[嘉宾姓名]老师，您好！

再次打扰，关于[活动名称]的嘉宾邀请，不知道您是否已有考虑？

如果时间不方便，我们也完全理解。感谢您的时间！
```

**确认参加后的感谢**：
```
[嘉宾姓名]老师，您好！

非常感谢您接受我们的邀请！接下来我会与您对接以下细节：

1. 演讲主题和PPT（请在X月X日前提供）
2. 个人简介和照片（用于活动宣传）
3. 行程安排（到达/离开时间）
4. 接待需求（住宿/交通/饮食禁忌）

我会在近期与您详细沟通。再次感谢您的支持！
```

## 第五步：收集嘉宾信息

### 嘉宾信息收集表

| 字段 | 说明 | 示例 |
|------|------|------|
| 姓名 | 全名 | 张三 |
| 职位/头衔 | 当前职位 | XX公司CTO |
| 个人简介 | 用于活动宣传，100字以内 | 张三，XX公司CTO，10年技术管理经验... |
| 照片 | 用于宣传物料，高清正面照 | - |
| 演讲主题 | 分享/演讲的主题 | AI时代的创业机会 |
| 演讲PPT | 路演用PPT | PDF/PPTX格式 |
| 到达时间 | 何时到达活动城市 | 6/15 14:00 |
| 离开时间 | 何时离开 | 6/16 18:00 |
| 差旅信息 | 航班/高铁信息 | G1234 北京南-上海虹桥 |
| 住宿需求 | 是否需要安排住宿 | 是，大床房 |
| 饮食禁忌 | 特殊饮食需求 | 无 |
| 联系方式 | 手机+微信 | 138xxxx |
| 其他需求 | 其他特殊需求 | 需要投影+翻页笔 |

### 信息收集时间节点

- **确认参加后**：收集基本信息（姓名、职位、简介、照片）
- **活动前3周**：收集演讲主题
- **活动前2周**：收集演讲PPT
- **活动前1周**：收集行程和接待信息

## 常见问题

**Q：嘉宾要求出场费怎么办？**
A：①先确认预算是否允许；②可以提供其他权益代替（如活动曝光、社群资源）；③如果预算有限，优先邀请不需要出场费的嘉宾；④赞助商嘉宾通常不需要出场费。

**Q：嘉宾临时取消怎么办？**
A：①立即启动备选嘉宾方案；②调整日程安排；③提前准备每位嘉宾的备选方案；④保持与嘉宾的密切沟通，尽早发现问题。

**Q：如何邀请行业大咖？**
A：①通过熟人引荐（成功率最高）；②在社交媒体上私信（附上活动介绍）；③通过赞助商/合作方推荐；④如果直接邀请困难，可以先邀请其团队成员。

**Q：线上嘉宾和线下嘉宾如何安排？**
A：①优先邀请线下嘉宾（体验更好）；②对于时间紧张或外地大咖，提供线上参与选项；③线上嘉宾需要提前测试设备和网络。

## 历史教训

在hackathon活动中，嘉宾邀请建议：
- **导师比大咖更实用**：hackathon选手更需要技术导师的现场指导，而不是大咖的演讲
- **行业多样性**：邀请不同行业/技术方向的导师，覆盖选手可能的项目方向
- **往届嘉宾**：往届合作过的嘉宾是最可靠的邀请来源
- **嘉宾时间**：hackathon通常持续1-2天，不需要嘉宾全程在场，只需在特定环节参与
- **学生嘉宾**：邀请往届获奖选手分享经验，对新手选手很有参考价值
