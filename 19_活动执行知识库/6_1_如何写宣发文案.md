# 6_1 如何写宣发文案

> 适用场景：活动各阶段需要撰写宣发文案，包括预热、招募、活动期间和传播期的各类文案。
> 前置条件：[06_宣发与品牌传播](06_宣发与品牌传播.md)

---

## 第一步：了解不同阶段的文案类型

| 阶段 | 文案类型 | 目的 | 主要渠道 |
|---|---|---|---|
| **预热期**（T-4~2周） | 悬念海报、预告推文 | 吸引关注、制造期待 | 微信、小红书、朋友圈 |
| **招募期**（T-3~1周） | 选手招募、展商招募、志愿者招募 | 吸引报名 | 全渠道 |
| **活动期**（T日） | 活动现场播报、实时图文 | 扩大传播 | 社群、朋友圈、小红书 |
| **传播期**（T+1~2周） | 活动回顾、精彩瞬间、媒体报道 | 沉淀内容、品牌传播 | 全渠道 |

---

## 第二步：写预热期文案

### 预热海报文案模板

```
【悬念式】

[日期]
一场关于 [主题关键词] 的 [活动类型] 即将开启

[一句悬念语，如：你准备好迎接挑战了吗？]

📍 [城市]
📅 [日期]
🔗 详情即将公布，敬请期待

#活动名称 #关键词1 #关键词2
```

### 预热推文模板

```
标题：[悬念/疑问句，吸引点击]
      如："XX城市最酷的周末，你来不来？"

正文：
开头（制造期待）：
[用1-2句话描述活动的亮点或独特之处]

中间（透露信息）：
📅 时间：[日期]
📍 地点：[地点]
🎯 主题：[主题]
👥 规模：[人数]

结尾（行动号召）：
想了解更多？关注我们，[日期] 正式开放报名！
[关注/转发引导]

配图建议：
- 主视觉海报 1张
- 往期活动照片 2-3张（如有）
```

## 第三步：写招募期文案

### 选手招募文案模板

```
标题：[活动名称] 选手招募 | [一句吸引人的话]
      如："48小时，把你的想法变成现实"

正文：

🎯 这是什么？
[用2-3句话介绍活动]

💡 为什么参加？
- [亮点1，如：与XX位行业大咖面对面]
- [亮点2，如：总奖金池XX元]
- [亮点3，如：往期XX+项目成功孵化]
- [亮点4，如：认识志同道合的伙伴]

📋 活动信息
- 时间：[日期]
- 地点：[地点]
- 规模：[人数]
- 费用：[免费/XX元]

📝 报名方式
[报名链接/二维码]
截止日期：[日期]

❓ 常见问题
Q：需要什么技能？
A：[回答]
Q：可以一个人报名吗？
A：[回答]

📞 联系方式
[微信号/邮箱/电话]
```

### 展商/合作方招募文案模板

```
标题：[活动名称] 展位/合作招募中！

正文：

🎪 活动概况
- 预计参与人数：XX人
- 活动时间：[日期]
- 活动地点：[地点]

🎁 展商/合作权益
- ✅ 现场XX人次曝光
- ✅ 官方全渠道宣传露出
- ✅ [其他权益]

📊 往期数据（如有）
- 参与人数：XX
- 社交媒体曝光：XX
- 合作方满意度：XX%

📝 报名方式
[报名链接/二维码]
联系人：[姓名] [微信号]
```

```
#活动名称 #关键词
```

### 小红书活动期文案模板

```

✨ 亮点3：[描述]

#活动名称 #黑客松 #城市名 #关键词
```

```
[赞助商logo展示]
```

## 第六步：标题技巧

### 标题公式

| 公式 | 示例 |
|---|---|
| **数字+关键词** | "48小时，XX个创意项目诞生了" |
| **疑问句** | "参加黑客松到底值不值？" |
| **悬念式** | "这个周末，XX城市有一件大事发生" |
| **对比式** | "从想法到产品，只用了48小时" |
| **情感式** | "这是我参加过最燃的活动" |
| **利益式** | "免费参加+XX奖金，这个黑客松太香了" |

### 标题避坑

| ❌ 不要这样 | ✅ 应该这样 |
|---|---|
| "活动通知" | "你的周末计划，我们承包了" |
| "XX活动开始报名" | "48小时，把你的创意变成现实" |
| "关于我们" | "一群创客的48小时狂欢" |

## 第七步：配图要求

### 配图规范

| 渠道 | 尺寸 | 格式 | 要求 |
|---|---|---|---|
| 微信公众号头图 | 900 × 383px | JPG/PNG | 文字清晰、主题突出 |
| 微信公众号文内图 | 宽度900px | JPG/PNG | 与内容相关 |
| 小红书封面 | 1080 × 1440px | JPG/PNG | 竖版、有视觉冲击 |
| 小红书文内图 | 1080 × 1080px | JPG/PNG | 正方形、清晰 |
| 朋友圈海报 | 1080 × 1920px | JPG/PNG | 竖版、信息完整 |
| 横版海报 | 1920 × 1080px | JPG/PNG | 用于大屏/投影 |

### 配图 Checklist

- [ ] 图片清晰度足够（不低于72dpi，印刷品不低于300dpi）
- [ ] 文字在图片上可读（手机端能看清）
- [ ] 品牌元素一致（Logo、字体、颜色）
- [ ] 无版权问题（使用授权图片或原创拍摄）
- [ ] 每篇文案至少配1张封面图

## 第八步：发布渠道选择

| 渠道 | 适合发布什么 | 发布时间建议 | 阅读高峰 |
|---|---|---|---|
| 微信公众号 | 深度推文、活动回顾 | 中午12点 / 晚上8点 | 工作日午休、周末 |
| 小红书 | 精美图文、活动体验 | 晚上7-10点 | 周末 |
| 朋友圈 | 短文案+图片 | 随时 | 早中晚高峰 |
| 微信社群 | 活动通知、实时播报 | 即时 | — |
| 微博 | 热点话题、实时动态 | 上午10点 / 晚上8点 | 工作日 |
| 抖音/B站 | 活动视频、Vlog | 晚上6-9点 | 周末 |

## 第九步：审核流程

### 文案审核流程

| 步骤 | 操作 | 负责人 | 时间要求 |
|---|---|---|---|
| 1 | 撰写初稿 | 宣发文案 | 发布前3天 |
| 2 | 内容审核（事实、数据准确性） | 宣发负责人 | 发布前2天 |
| 3 | 品牌审核（视觉、调性一致性） | 设计负责人 | 发布前2天 |
| 4 | 最终确认 | 总负责人 | 发布前1天 |
| 5 | 定时发布 | 宣发组 | 按计划时间 |

### 文案审核 Checklist

- [ ] 信息准确（时间、地点、人名、数据）
- [ ] 无错别字和语法错误
- [ ] 链接和二维码可正常使用
- [ ] 图片清晰、无水印
- [ ] 符合品牌调性
- [ ] 无敏感内容
- [ ] 有明确的行动号召（CTA）

## 常见问题

**Q：文案写不出彩怎么办？**
A：①多参考同类活动的宣发文案；②找3个关键词作为核心卖点反复强调；③用真实故事或数据代替空洞描述；④让团队成员一起头脑风暴。

**Q：阅读量/互动量低怎么办？**
A：①检查标题是否有吸引力；②调整发布时间到阅读高峰期；③增加互动引导（提问、投票、抽奖）；④在社群中转发引导；⑤请合作方帮忙转发。

**Q：宣发预算有限怎么办？**
A：①优先做免费渠道（社群、朋友圈、小红书自然流量）；②设计好转发裂变机制（如转发抽门票）；③找KOL/博主做资源置换（免费参加换宣传）；④集中预算在1-2个核心渠道。

**Q：内容被投诉或有负面评论怎么办？**
A：①及时回复，态度诚恳；②如果确实是问题，快速修正并道歉；③如果是恶意评论，不回复不扩散；④重要情况及时上报总负责人。

## 历史教训

> 历史活动的经验：宣发文案需要提前规划好内容日历，不同阶段发布不同类型的内容。招募期文案是最关键的，标题决定了打开率，建议准备3-5个标题让团队投票选择。小红书的配图非常重要，竖版图片效果最好，封面图要有视觉冲击力。活动期间的实时播报要安排专人负责，现场拍照+15分钟内发出去。所有文案发布前必须经过审核，确保信息准确无误。合作方的宣发文案要提前发给对方确认，并提供统一的素材包（图片、文案、话题标签）。
