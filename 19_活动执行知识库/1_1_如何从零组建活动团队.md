# 1_1 如何从零组建活动团队

> 适用场景：你决定要办一场活动，现在需要组建一个能干活的团队。
> 前置条件：无

---

## 第一步：确定你需要多少人

根据活动规模估算：

| 你的活动规模 | 参与人数 | 你需要的团队人数 |
|---|---|---|
| 小型（校园活动） | 30-100人 | 8-15人 |
| 中型（城市级活动） | 100-300人 | 15-30人 |
| 大型（全国级活动） | 300-500人 | 30-50人 |

---

## 第二步：确定你需要哪些岗位

照着下面的表，把每个岗位填上具体的人名：

| 岗位 | 需要几个人 | 这个人需要会什么 | 备注 |
|---|---|---|---|
| **总负责人** | 1人 | 能拍板、能协调、抗压能力强 | 最重要的角色 |
| **副总负责人** | 1人 | 能补位总负责人 | 总负责人不在时顶上 |
| **策划负责人** | 1人 | 有活动策划经验 | 负责活动方案 |
| **设计负责人** | 1人 | 会PS/Figma/Canva | 负责所有视觉设计 |
| **宣发负责人** | 1人 | 熟悉社交媒体运营 | 负责推广和传播 |
| **商务负责人** | 1人 | 沟通能力强、有商务经验 | 负责赞助和合作 |
| **技术负责人** | 1人 | 懂网站/设备/网络 | 负责线上系统和现场设备 |
| **财务负责人** | 1人 | 细心、会用Excel | 负责预算和报销 |
| **志愿者负责人** | 1人 | 有管理经验 | 负责志愿者招募和管理 |

---

## 第三步：去哪里找人

### 找人的渠道

| 渠道 | 适合找什么人 | 怎么操作 |
|---|---|---|
| **身边朋友** | 核心团队 | 先找最信任的人聊，邀请加入 |
| **高校社团/学生会** | 执行人员 | 联系社团负责人，说明活动情况 |
| **社交媒体** | 志愿者、兼职 | 发招募帖（小红书、朋友圈、微信群） |
| **往届活动参与者** | 有经验的人 | 联系参加过类似活动的人 |
| **合作方推荐** | 专业人员 | 请赞助商/合作方推荐 |

### 面试/沟通要点

跟候选人聊的时候，确认以下几点：

1. **时间承诺**：活动前X周，每周能投入多少小时？
2. **技能匹配**：这个岗位需要的技能，TA会不会？
3. **过往经验**：以前有没有做过类似的事情？
4. **责任心**：TA能不能在截止时间前完成任务？
5. **团队协作**：TA跟其他人配合得来吗？

---

## 第四步：建立协作机制

团队组建完成后，立即做以下几件事：

### 4.1 建群

- 建立核心团队微信群/飞书群
- 群名称格式：【活动名称】核心团队
- 群公告写清楚：团队分工、紧急联系方式

### 4.2 第一次全员会议

会议议程（建议1小时）：

```
1. 活动背景介绍（10分钟）
   - 我们要办什么活动？
   - 为什么办？
   - 什么时候办？
2. 团队介绍（10分钟）
   - 每个人自我介绍
   - 每个人负责什么
3. 时间线确认（15分钟）
   - 关键节点有哪些？
   - 每个节点谁负责？
4. 协作方式确认（15分钟）
   - 用什么工具协作？（飞书/Notion/微信群）
   - 多久开一次会？
   - 文档存放在哪里？
5. 自由讨论（10分钟）
   - 大家有什么问题？
```

### 4.3 建立文档协作空间

在飞书/Notion中创建以下文件夹：

```
活动名称/
├── 01_活动方案/
├── 02_预算/
├── 03_设计素材/
├── 04_宣发文案/
├── 05_嘉宾信息/
├── 06_选手信息/
├── 07_物料清单/
├── 08_会议记录/
└── 09_其他/
```

---

## 第五步：明确分工规则

写一份简单的分工协议，让每个人签字确认：

```
【分工确认书】

我，[姓名]，确认担任[活动名称]的[岗位名称]，负责以下工作：
1. [具体职责1]
2. [具体职责2]
3. [具体职责3]

我承诺：
- 在截止时间前完成任务
- 遇到问题及时沟通
- 服从团队统一安排

签名：____________
日期：____________
```

## 常见问题

**Q：找不到合适的人怎么办？**
A：先从身边最信任的人开始，哪怕能力不够但责任心强的人，比能力强但不靠谱的人好。能力可以培养，态度很难改变。

**Q：团队成员中途退出怎么办？**
A：提前约定退出机制（如提前X周通知），关键岗位设置AB角（备份人员）。

**Q：团队内部有矛盾怎么处理？**
A：先私下沟通，了解双方诉求。如果无法解决，由总负责人裁决。活动期间以任务为重，个人情绪放到活动后处理。

### 真实案例：历史活动团队分工表

| 事项 | 总负责人 | 分负责部分 |
|---|---|---|
| 4.30阿里场地布置 | 总负责人 | 场地规划、桌椅、帐篷、展商布置 |
| 4.30酒会布置 | 策划负责人 | 物料清单+运送、物料布置 |
| 4.30开幕式布置 | 场地负责人 | 场地B1F+3F物料布置 |
| 4.30开幕式彩排 | 宣发负责人 | PPT、音频、视频、话筒、机位 |
| 4.30酒会活动 | 负责人E | 签到、吧台、主持+开放麦 |
| 酒店负责 | 负责人F | 嘉宾/选手住宿对接 |
| 后厨负责 | 负责人G | 餐饮对接、餐券管理 |
| 财务部分 | 负责人H | 预算、报销、台账 |
| 机动组 | 负责人I/J/K | 至少2-3人常驻开发场地 |

**关键教训**：分工文档不要搞太多，集中在一个地方维护。总负责人（2人）负责盯紧各组进度、调配人员、应急决策。

---

## 历史教训

> 历史活动的核心教训：分工文档不要搞太多，集中在一个地方维护。总负责人（总负责人、策划负责人）负责盯紧各组进度、调配人员、应急决策。
