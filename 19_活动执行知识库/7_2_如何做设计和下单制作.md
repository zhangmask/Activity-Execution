# 如何做设计和下单制作

> **适用场景**：活动物料清单确定后，需要进行视觉设计、审核校对并下单制作。
> **前置条件**：已完成物料清单和供应商选择（参考 7_1_如何列物料清单和找供应商）。

---

## 第一步：编写设计需求文档

在找设计师之前，先写清楚需求，避免反复修改。

### 设计需求文档模板

```
【设计需求文档】

一、项目信息
- 活动名称：____________________
- 活动日期：____________________
- 设计需求发起人：______________
- 期望交付日期：________________

二、设计内容清单
| 序号 | 物料名称 | 尺寸 | 数量 | 用途说明 |
|------|----------|------|------|----------|
| 1 | 主KV背景板 | 3m×2.5m | 1 | 签到区背景 |
| 2 | 易拉宝画面 | 80cm×200cm | 6 | 各区域指引 |

三、设计风格要求
- 主色调：______________________
- 参考风格：（附参考图链接）
- 必须包含的元素：logo、slogan、二维码
- 必须包含的文字内容：（逐条列出）

四、交付要求
- 文件格式：AI/PSD源文件 + PNG/JPG成品
- 分辨率：印刷类300dpi，屏幕类72dpi
- 色彩模式：印刷用CMYK，屏幕用RGB

五、修改约定
- 包含____次免费修改
- 修改响应时间：____小时内
```

## 第二步：主视觉延展设计

### 什么是主视觉？

主视觉（主KV）是活动的核心视觉形象，所有物料的设计都基于主KV进行延展。

### 主视觉延展流程

```
主KV设计确认
    ↓
延展物料清单梳理
    ↓
按优先级逐一延展
    ↓
统一审核
    ↓
输出印刷文件
```

### 主KV设计要素

| 要素 | 说明 | 注意事项 |
|------|------|----------|
| 活动名称 | 中英文全称 | 字体大小突出 |
| 活动主题/slogan | 一句话概括 | 简洁有力 |
| 时间地点 | 具体日期和场地 | 不要写错！ |
| 主办方logo | 所有赞助方/主办方 | 按协议排列顺序 |
| 主视觉图形 | 核心视觉元素 | 统一风格 |
| 报名二维码 | 链接到报名页面 | 确保可扫码 |

### 延展优先级

1. **必须有**：背景板、易拉宝、胸牌、PPT模板
2. **建议有**：指引牌、社交媒体banner、手提袋
3. **锦上添花**：贴纸、拍照框、桌面立牌

## 第三步：审核设计稿

### 设计稿审核要点checklist

**文字内容审核**：
- [ ] 活动名称是否正确（中英文）
- [ ] 日期是否正确（年月日、星期几）
- [ ] 时间是否正确（24小时制）
- [ ] 地点是否正确（场馆全称+地址）
- [ ] 嘉宾姓名和头衔是否正确
- [ ] 赞助方logo是否齐全且正确
- [ ] 二维码是否可正常扫码
- [ ] 英文是否有拼写错误
- [ ] 联系方式是否正确

**视觉审核**：
- [ ] 颜色是否符合品牌规范
- [ ] 字体是否统一
- [ ] 图片是否清晰（无像素感）
- [ ] 排版是否整齐美观
- [ ] 重要信息是否醒目

**技术审核**：
- [ ] 文件尺寸是否正确
- [ ] 出血位是否预留（一般3mm）
- [ ] 分辨率是否达标（印刷300dpi）
- [ ] 色彩模式是否正确（CMYK）

## 第四步：印刷前校对

### 印刷前校对checklist

这是最后一道关卡，务必认真执行！

**第一步：自己校对**
- 打印一份A4小样
- 逐字逐句对照原始信息
- 用尺子检查对齐

**第二步：交叉校对**
- 换一个同事校对（新鲜的眼睛更容易发现错误）
- 重点检查：数字、日期、人名、英文

**第三步：实物打样确认**
- 要求供应商出一份实物打样
- 核对颜色、材质、尺寸
- 确认无误后签字确认批量生产

### 常见印刷错误清单

| 错误类型 | 案例 | 检查方法 |
|----------|------|----------|
| 日期错误 | 6月16日写成6月15日 | 对照日历核对 |
| 时间错误 | 上午9点写成下午9点 | 使用24小时制 |
| 地点错误 | 楼层/房间号写错 | 对照场地合同 |
| 人名错误 | 嘉宾名字写错 | 让嘉宾本人确认 |
| 英文拼写 | "Hackathon"写错 | 使用拼写检查工具 |
| 二维码失效 | 链接已过期 | 每次都重新扫码测试 |

## 第五步：下单制作

### 下单时间线

| 物料类型 | 建议下单时间（活动前） | 生产周期 | 备注 |
|----------|----------------------|----------|------|
| 定制T恤 | 3周前 | 7-15天 | 需提前确认尺码 |
| 定制徽章/贴纸 | 3周前 | 7-10天 | 批量生产 |
| 背景板/易拉宝 | 2周前 | 3-5天 | 本地制作 |
| 胸牌/工作证 | 2周前 | 3-5天 | 含挂绳 |
| 奖品采购 | 2周前 | 3-7天 | 确认库存 |
| 指引牌/KT板 | 1周前 | 1-3天 | 本地制作 |
| 手提袋 | 3周前 | 7-10天 | 丝印/热转印 |

### 下单确认checklist

下单前确认以下内容：
- [ ] 尺寸规格确认无误
- [ ] 数量确认（建议多备5%-10%余量）
- [ ] 颜色确认（要求供应商提供色号或打样）
- [ ] 交付日期确认（留出1-2天缓冲）
- [ ] 交付地址确认
- [ ] 价格和付款方式确认
- [ ] 是否含税/开发票
- [ ] 退换货条款确认

### 打样确认流程

```
提交设计文件给供应商
    ↓
供应商出打样（拍照/实物）
    ↓
核对颜色、材质、尺寸、印刷质量
    ↓
如有问题 → 标注修改意见 → 重新打样
    ↓
确认无误 → 签字确认 → 批量生产
    ↓
批量到货 → 验收检查
```

## 常见问题

**Q：没有设计师怎么办？**
A：可以使用Canva、稿定设计等在线工具套模板修改；也可以在闲鱼、猪八戒上找兼职设计师，一套物料设计费通常在500-2000元。

**Q：设计稿颜色和印出来不一样怎么办？**
A：这是CMYK和RGB色域差异导致的。要求设计师使用CMYK模式，印刷前务必要求实物打样确认。

**Q：临时需要加印怎么办？**
A：提前与供应商沟通紧急加印的可能性和费用。本地广告公司通常可以当天或次日交付简单印刷品。

**Q：如何控制设计修改次数？**
A：在需求文档中明确修改次数（一般3次），每次修改提供完整的修改意见汇总，避免零散反馈。

## 历史教训

在hackathon活动中，设计和制作建议：
- **主KV尽早确定**：主视觉是所有物料的基础，建议在活动前4周确定
- **模板化思维**：用统一的PPT模板、邮件模板、海报模板，保持视觉一致性
- **校对铁律**：日期、时间、地点这三项必须交叉校对至少3次
- **设计师沟通**：如果找外部设计师，一次把所有需求和参考图给齐，减少沟通成本
- **应急方案**：准备一份通用的可编辑海报模板，现场需要临时制作物料时可以直接修改打印
