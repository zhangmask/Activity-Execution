# 3_1 如何选场地和踩点

> 适用场景：需要为活动找一个合适的场地。
> 前置条件：[1_2_如何制定活动时间线](1_2_如何制定活动时间线.md)

---

## 第一步：明确你的场地需求

在找场地之前，先回答以下问题：

| 问题 | 你的答案 |
|---|---|
| 活动时间是什么？ | xx月xx日-xx日 |
| 预计多少人参加？ | xxx人 |
| 需要哪些功能区？ | 主会场/开发区/展区/休息区 |
| 预算多少？ | xxx元 |
| 在哪个城市/区域？ | xxx |
| 交通要求？（地铁/公交/停车） | xxx |
| 需要住宿吗？ | 是/否 |
| 需要餐饮吗？ | 是/否 |

---

## 第二步：去哪里找场地

| 渠道 | 适合找什么场地 | 怎么操作 |
|---|---|---|
| **高校场地** | 学校报告厅、体育馆 | 联系学校后勤/场地管理部门 |
| **产业园区** | 创业园区、孵化器 | 联系园区管理处 |
| **会议中心** | 专业会议场地 | 搜索"xx城市会议中心" |
| **酒店** | 酒店会议室 | 联系酒店销售部 |
| **商业地产** | 商场活动空间 | 联系商场运营部 |
| **朋友推荐** | 各种场地 | 问问身边人有没有资源 |

---

## 第三步：筛选候选场地

从找到的场地中，按以下标准筛选出2-3个候选：

| 筛选标准 | 必须满足 | 最好满足 |
|---|---|---|
| 容纳人数 | >= 你的预计人数 | 1.5倍余量 |
| 交通 | 有公共交通到达 | 地铁站附近 |
| 价格 | 在预算范围内 | 包含基础设备 |
| 时间 | 活动日期可用 | 能提前1天进场 |
| 设备 | 有基本的音响/投影 | 有LED屏 |

---

## 第四步：去现场踩点

**踩点是必须的！不能只看照片和参数。**

### 踩点 Checklist

带着这个清单去现场，逐项确认：

#### 基础条件
- [ ] 实际容纳人数（坐下来能坐多少人）
- [ ] 层高（影响搭建和视觉效果）
- [ ] 柱子/障碍物（是否遮挡视线）
- [ ] 采光（自然光是否充足）
- [ ] 通风/空调（温度能否控制）

#### 设备条件
- [ ] LED屏/投影（尺寸、分辨率、接口类型）
- [ ] 音响系统（功率、话筒数量）
- [ ] 灯光（舞台灯光、环境灯光）
- [ ] 网络（WiFi承载人数、有线网络端口）
- [ ] 电源（插座数量、总功率容量）

#### 配套设施
- [ ] 卫生间数量和位置
- [ ] 电梯/货梯（大件物品搬运）
- [ ] 停车场（车位数量、停车费）
- [ ] 餐饮供应（是否有食堂/厨房）
- [ ] 住宿条件（是否有配套酒店/宿舍）

#### 安全条件
- [ ] 消防通道
- [ ] 灭火器
- [ ] 紧急出口
- [ ] 安保服务

### 踩点时要做的事

1. **拍照/录像**：记录场地的每个区域
2. **测量尺寸**：主会场、展区、签到区的长宽
3. **测试设备**：试一下投影、音响、网络
4. **了解对接人**：确认场地方面的对接人和联系方式
5. **了解内部流程**：进场需要什么手续？需要提前多久申请？

## 第六步：确认场地

选定场地后：

1. **签合同**：签场地租赁合同（参考[2_3_如何签合同](2_3_如何签合同.md)）
2. **支付定金**：按合同约定支付定金
3. **确认对接人**：获取场地对接人的联系方式
4. **确认进场时间**：确认可以提前多久进场布置
5. **申请门禁/停车**：如需要，提前申请

## 常见问题

**Q：找不到合适的场地怎么办？**
A：扩大搜索范围（考虑郊区、周边城市），或者降低非核心需求（如不住宿、不提供餐饮）。

**Q：场地价格超出预算怎么办？**
A：跟场地谈价格（长期合作、打包优惠），或者找赞助商赞助场地费用。

**Q：场地有多个，怎么选？**
A：按重要性排序：安全 > 容量 > 交通 > 设备 > 价格。安全性是底线。

## 历史教训

> 历史活动的场地：采用场地A（主开发区+展区）+ 场地B（开幕式+游园会+路演）的双会场模式。教训是场地确认应尽早，场地不定则后续所有工作都无法推进。
