```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                        获 奖 证 书                           │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│    [获奖者姓名/团队名称]：                                    │
│                                                             │
│    您在 [活动名称] 中表现优异，                               │
│                                                             │
│    特授予 [奖项名称]。                                        │
│                                                             │
│    项目名称：[项目名称]                                       │
│                                                             │
│    颁发单位：[主办方名称]                                     │
│                                                             │

```

---

## 设计建议

### 版式布局

- **尺寸**：A4横向（297mm × 210mm）
- **边距**：上下左右各20mm
- **字体**：标题使用书法体或艺术字，正文使用宋体或微软雅黑
- **颜色**：金色、深蓝色、红色为主色调

### 元素建议

- **边框**：金色或深蓝色边框
- **徽标**：活动LOGO放在左上角或居中
- **印章**：主办方电子印章放在右下角
- **签名**：负责人签名放在右下角

### 制作工具

- **推荐**：Canva、Figma、Photoshop
- **模板**：可在Canva搜索"证书模板"获取灵感

---

## 打印建议

### 纸张选择

- **推荐**：250g铜版纸或特种纸
- **效果**：哑光或亮光均可
- **尺寸**：A4标准尺寸

### 打印注意事项

1. **颜色校准**：打印前先打样确认颜色
2. **裁切**：预留3mm出血线
3. **数量**：多准备10%备用
4. **检查**：打印后逐张核对姓名和奖项

## 使用说明

1. 将方括号 [] 中的内容替换为实际信息
2. 根据活动风格调整设计元素
3. 打印前务必核对所有信息
4. 建议提前3天完成设计和打印
