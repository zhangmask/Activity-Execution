# 志愿者报名问卷字段设计

> 用于收集志愿者信息，建议使用飞书表单或腾讯问卷制作

---

## 基础信息（必填）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 姓名 | 文本 | 真实姓名 |
| 性别 | 单选 | 男/女 |
| 手机号 | 数字 | 11位手机号 |
| 微信号 | 文本 | 用于拉群 |
| 所在城市 | 文本 | 确认是否需要住宿 |
| 学校/单位 | 文本 | 在读学校或工作单位 |

## 可用时间（必填）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 可参加日期 | 多选 | Day1/Day2/Day3/全部 |
| 可参加时段 | 多选 | 上午/下午/晚上/全天 |
| 是否可参加培训 | 单选 | 是/否 |
| 是否可参加彩排 | 单选 | 是/否 |

## 岗位意向（必填）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 首选岗位 | 单选 | 签到组/引导组/物料组/机动组/技术组 |
| 次选岗位 | 单选 | 同上 |
| 相关经验 | 长文本 | 之前做过的志愿者工作 |
| 特长技能 | 多选 | 沟通能力/技术能力/设计能力/体力好/细心 |

## 其他信息（选填）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| T恤尺码 | 单选 | S/M/L/XL/XXL |
| 饮食禁忌 | 文本 | 素食/过敏等 |
| 特殊需求 | 文本 | 无障碍需求等 |
| 对活动的期待 | 长文本 | 选填 |

---

## 问卷设置建议

1. **必填字段**：标红或加星号
2. **逻辑跳转**：选择特定岗位后显示相关问题
3. **筛选条件**：优先选择可全程参加的志愿者
4. **截止时间**：活动前2周截止报名
5. **确认邮件**：报名成功后自动发送确认邮件
