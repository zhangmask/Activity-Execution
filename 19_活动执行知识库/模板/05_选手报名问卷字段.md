# 选手报名问卷字段设计

> 用于收集参赛选手信息，建议使用飞书表单或腾讯问卷制作

---

## 基础信息（必填）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 姓名 | 文本 | 真实姓名 |
| 性别 | 单选 | 男/女 |
| 手机号 | 数字 | 11位手机号 |
| 微信号 | 文本 | 用于拉群 |
| 邮箱 | 文本 | 接收通知 |
| 所在城市 | 文本 | 便于安排住宿 |
| 学校/单位 | 文本 | 在读学校或工作单位 |
| 专业/岗位 | 文本 | 技术方向 |

## 技术背景（必填）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 技术栈 | 多选 | 前端/后端/AI/设计/产品/其他 |
| 编程经验 | 单选 | 1年以下/1-3年/3-5年/5年以上 |
| 参赛经历 | 单选 | 首次参赛/1-2次/3次以上 |
| GitHub链接 | 文本 | 选填 |
| 个人作品集 | 文本 | 选填 |

## 参赛意向（必填）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 想参加的赛道 | 单选/多选 | 根据活动设置 |
| 是否已组队 | 单选 | 是/否 |
| 团队名称 | 文本 | 已组队填写 |
| 团队成员 | 文本 | 已组队填写其他成员姓名 |
| 期望队友技术栈 | 多选 | 希望队友具备的技术 |
| 项目想法 | 长文本 | 简述想做的项目方向 |

## 其他信息（选填）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 饮食禁忌 | 文本 | 素食/过敏等 |
| 特殊需求 | 文本 | 无障碍需求等 |
| 如何得知活动 | 单选 | 朋友推荐/社交媒体/学校/其他 |
| 对活动的期待 | �文本 | 选填 |

---

## 问卷设置建议

1. **必填字段**：标红或加星号
2. **逻辑跳转**：已组队才显示团队信息字段
3. **字数限制**：项目想法限制300字以内
4. **截止时间**：设置明确的报名截止日期
5. **确认邮件**：报名成功后自动发送确认邮件
