### OBS场景配置

#### 场景1：主讲模式

**用途**：嘉宾演讲、路演展示

**画面构成**：
- 主画面：嘉宾/PPT全屏
- 画中画：嘉宾摄像头（右下角）
- 底部条：活动名称、嘉宾信息

**设置**：
- 分辨率：1920x1080
- 帧率：30fps
- 码率：4000kbps

#### 场景2：PPT + 画中画

**用途**：PPT展示、项目介绍

**画面构成**：
- 主画面：PPT全屏
- 画中画：演讲者摄像头（右下角）
- 底部条：活动名称、项目信息

**设置**：
- 分辨率：1920x1080
- 帧率：30fps
- 码率：4000kbps

#### 场景3：等待画面

**用途**：活动开始前、休息时间

**画面构成**：
- 主画面：活动海报/倒计时
- 底部条：活动名称、开始时间

**设置**：
- 分辨率：1920x1080
- 帧率：30fps
- 码率：2000kbps

#### 场景4：多机位

**用途**：开幕式、颁奖典礼

**画面构成**：
- 主画面：主讲人
- 副画面：观众反应
- 底部条：活动名称

**设置**：
- 分辨率：1920x1080
- 帧率：30fps
- 码率：4000kbps

### 直播流程

#### 直播前（T-1天）

1. [ ] 测试网络带宽
2. [ ] 测试推流地址
3. [ ] 测试摄像头
4. [ ] 测试话筒
5. [ ] 测试灯光
6. [ ] 配置OBS场景
7. [ ] 测试场景切换
8. [ ] 准备备用设备

#### 直播当天（T日）

1. [ ] 提前2小时到场
2. [ ] 连接设备
3. [ ] 测试网络
4. [ ] 测试推流
5. [ ] 测试画面
6. [ ] 测试声音
7. [ ] 开始推流
8. [ ] 监控直播

#### 直播中

1. [ ] 监控画面质量
2. [ ] 监控声音质量
3. [ ] 监控网络状态
4. [ ] 切换场景
5. [ ] 处理弹幕互动
6. [ ] 记录问题

#### 直播后

1. [ ] 停止推流
2. [ ] 保存直播回放
3. [ ] 备份直播素材
4. [ ] 整理设备
5. [ ] 总结问题

### 常见问题处理

| 问题 | 解决方案 |
|------|----------|
| 画面卡顿 | 降低码率、检查网络 |
| 声音不同步 | 调整音频延迟 |
| 画面黑屏 | 检查摄像头连接 |
| 推流失败 | 检查推流地址、网络 |
| 弹幕延迟 | 调整弹幕设置 |
