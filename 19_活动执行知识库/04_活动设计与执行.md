# 04 活动设计与执行

> 本手册适用于活动各环节的设计、流程编排和现场执行。
> 参考案例：活动主办方 历史活动（预热酒会 → 开幕式 → 开发 → Workshop → 游园会 → 路演 → 颁奖 → After Party）

---

## 一、核心原则

1. **活动方案是所有工作的总纲**：场地、物料、宣发、嘉宾全部围绕活动方案展开
2. **每个环节必须有唯一负责人**：环节负责人对该环节的流程、人员、物料负全责
3. **留出弹性时间**：每个环节之间预留15-30分钟缓冲，应对延误
4. **彩排是必须的**：所有涉及舞台、技术、流程的环节必须彩排

---

## 二、活动环节设计模板

### 2.1 典型黑客松活动流程

| 天数 | 上午 | 下午 | 晚上 |
|---|---|---|---|
| Day 0（预热日） | 场地布置 | 展位布置、开幕式彩排 | 预热酒会 |
| Day 1（开幕日） | 开幕式、展会开放 | Workshop、选手开发 | 嘉宾分享、闪电路演 |
| Day 2（开发日） | 选手开发、Workshop | 选手开发、导师辅导 | Demo测试 |
| Day 3（Demo Day） | 游园会、评委评分 | 路演、颁奖 | After Party |

### 2.2 单环节设计要素

每个环节的设计文档应包含：

```
环节名称：xxx
时间：xx:xx - xx:xx
地点：xxx
负责人：xxx
参与人数：xxx人
流程：
  1. xx:xx - xx:xx 环节A（具体内容）
  2. xx:xx - xx:xx 环节B（具体内容）
所需物料：xxx
所需设备：xxx
人员配置：xxx
应急预案：xxx
```

## 三、重点环节执行指南

### 3.1 开幕式

**关键流程**：签到 → 暖场视频 → 主持开场 → 嘉宾致辞 → 赛事发布 → 合影

**人员配置**：
- 签到组（2-3人）：签到表、胸牌发放、迟到信息同步
- 舞台执行（1人）：嘉宾候场、上下台衔接
- PPT负责人（1人）：主控PPT切换，提前收齐所有演讲PPT
- 音频负责人（1人）：BGM、话筒、视频声音
- 摄影摄像（2-3人）：机位安排、直播管理
- 秩序维护（2人）：门口、会场入口、爆满分流

**易遗漏**：
- 主持词中嘉宾姓名+title必须核实，提前给主持人确认
- 合影环节的站位组织、背景页设计
- 桌签+水提前摆放（桌签需包含所有嘉宾）
- 嘉宾伴手礼发放时机

### 3.2 游园会/展会

**关键流程**：展位布置 → 开放参观 → 评委评分 → 游客投票 → 收票统分

**评分机制设计**：
- 游客投票（贴纸制）：每人限投N票，防重复投票
- 评委评分：标准化评分表，明确维度和权重
- 综合计分：游客票×权重 + 评委分×权重 = 最终得分

**易遗漏**：
- 贴纸发放规则和展牌设计
- 评委KPI分配（每个评委负责多少个项目）
- 项目材料提交截止时间提醒机制
- 晋级名单通知流程

### 3.3 路演/路演

**关键流程**：团队候场 → PPT展示（5分钟） → Q&A（3分钟） → 评分

**人员配置**：
- 路演主持人：控制节奏、Q&A衔接
- 计时员：举牌/倒计时/强制结束
- PPT切换员：提前收集并测试所有路演PPT
- 计分员：实时汇总评委打分

**易遗漏**：
- 路演规则需提前详细说明（是否允许多人上台、设备使用等）
- 候场区的安排，避免选手看到前面的路演内容
- 评分表的电子化和备份

### 3.4 After Party

**关键流程**：场地布置 → 自由社交 → 表演/互动 → 散场

**易遗漏**：
- 如果涉及酒精饮品，需要确认场地许可
- 音乐/表演的版权问题
- 散场后的交通安排（深夜打车、大巴）
- 噪音控制（如果是室外场地）

## 四、评分与奖项设计

### 4.1 赛道设置

建议设置2-3条主赛道 + 可选的企业赞助赛道：
- 主赛道之间不可重复报名
- 企业赛道可与主赛道叠加申报
- 每条赛道独立设奖

### 4.2 奖项类型

| 奖项类型 | 说明 | 参考金额 |
|---|---|---|
| 赛道金奖 | 每条主赛道1个 | 3000-5000元 |
| 赛道银奖 | 每条主赛道1个 | 2000-3000元 |
| 企业赛道奖 | 由赞助商提供 | 赞助商定 |
| 社区人气奖 | 小红书点赞等社交指标 | 500-1000元 |
| 最佳协作奖 | 团队协作、持续运营 | 1000-2000元 |
| 全场大奖 | 所有项目中最佳 | 5000-10000元 |

### 4.3 评选流程

1. 游园会评分 → 筛选晋级名单
2. 路演展示 → 评委打分
3. 综合计算 → 确定获奖名单
4. 颁奖典礼

## 六、易遗漏事项

1. **环节间的转场时间**：不要把环节排得太紧，至少预留15-30分钟缓冲
2. **主持词撰写**：提前完成并审核，确保嘉宾信息准确
3. **音效设计**：不同环节使用不同风格BGM，明确每个环节的音乐清单
4. **倒计时工具**：路演等限时环节需要专业的倒计时工具
5. **线上评分系统**：提前测试，准备纸质备份方案
6. **蛋糕/仪式道具**：如果有切蛋糕等仪式环节，提前准备刀具等
7. **可乐墙/互动装置**：需要安排专门人员监督，可以合影但不能提前领取
8. **散场动线**：活动结束后的疏散路线和交通安排

## 八、历史教训

> **历史活动 活动方案**：采用"活动方案"文档作为所有事务的统筹分配中心，包含人员分工表、赛事核心流程表、物料表与预算、赛道奖项与评选等。每个环节都有明确的负责人。

> **历史活动 的经验**：开幕式彩排非常重要，PPT格式比例、音频、视频、话筒、摄影摄像机位都需要提前确认。所有电子文件（PPT、视频、BGM、主持词、流程表、座位图）必须提前收齐。
