# 如何做游客现场接待

> **适用场景**：黑客松活动对外开放日、Demo Day、展示日等需要接待外部游客的场景。
> **前置条件**：请先完成 `13_1_如何管理游客报名和门禁.md`，确认报名体系和门禁方案。

---

## 第一步：签到流程设计

### 标准签到流程

```
游客到达 → 排队等候 → 身份核验 → 领取手环 → 领取资料袋 → 入场引导
```

### 签到台布局方案

| 签到台编号 | 服务对象 | 配置 |
|-----------|---------|------|
| 1号台 | VIP/嘉宾 | 专人对接，快速通道 |
| 2号台 | 预报名观众（A-H姓氏） | 自助扫码 + 人工核验 |
| 3号台 | 预报名观众（I-Z姓氏） | 自助扫码 + 人工核验 |
| 4号台 | 现场报名 | 现场填表 + 核验 |
| 5号台 | 媒体 | 核验媒体证 + 发放紫色手环 |

### 签到台标准配置清单

- [ ] 签到表（纸质备份）
- [ ] 手机/平板（查看报名系统）
- [ ] 扫码枪或扫码APP
- [ ] 各颜色手环
- [ ] 资料袋（含活动手册、贴纸、小礼品）
- [ ] 签字笔（至少5支）
- [ ] 桌签（标注台号和服务对象）
- [ ] 垃圾桶
- [ ] 对讲机（1台）

### 快速签到话术

```
"您好！欢迎来到历史活动！请问您的姓名是？"（核验报名信息）
"好的，这是您的手环，请全程佩戴。这是您的资料袋。"
"主会场往前直走右手边，展示区在左手边。有任何问题可以找穿红色马甲的志愿者。"
"祝您参观愉快！"
```

## 第二步：游客引导方案

### 动线规划原则

1. **单向流动**：入口→签到→展示区→主会场→休息区→出口，避免人流交叉
2. **标识清晰**：每20米设置一个方向指引牌
3. **人+牌结合**：关键路口安排志愿者值守引导
4. **无障碍通道**：确保轮椅、婴儿车可通行

### 场地指引标识清单

| 位置 | 标识内容 | 数量 |
|------|---------|------|
| 入口处 | 活动主视觉 + 欢迎语 | 1个 |
| 签到区 | 各签到台编号和说明 | 每台1个 |
| 分岔路口 | 指向不同区域的方向牌 | 每个路口1个 |
| 楼梯/电梯 | 上下层区域说明 | 每处1个 |
| 卫生间 | 指引牌 | 每层至少1个 |
| 出口 | 出口标识 + 出口路线指引 | 2个 |

### 游客引导志愿者配置

| 位置 | 人数 | 职责 |
|------|------|------|
| 入口引导 | 2人 | 初步分流，引导至签到台 |
| 签到区引导 | 3人 | 协助签到，引导入场 |
| 场内流动引导 | 4人 | 回答问题，指引方向 |
| 展示区引导 | 2人 | 介绍展示项目 |
| 出口引导 | 1人 | 引导离场，收集反馈 |

## 第三步：游客互动设计

### 互动环节设计

| 互动方式 | 内容 | 物资准备 | 所需人力 |
|---------|------|---------|---------|
| 打卡集章 | 游客在各展示区盖章，集满可兑换礼品 | 集章卡、印章、礼品 | 每站1人 |
| 投票墙 | 游客为最喜欢的项目投票（贴纸） | 投票板、贴纸 | 1人维护 |
| 留言墙 | 游客写下感受或建议 | 便签纸、彩笔 | 1人维护 |
| 合影区 | 设置品牌背景板供游客拍照 | 背景板、道具 | 1人协助 |
| 体验区 | 让游客体验参赛项目 | 展示设备 | 每站1人 |

### 互动礼品建议

| 礼品 | 预算 | 用途 |
|------|------|------|
| 定制贴纸 | 2-5元/份 | 签到即送 |
| 定制徽章 | 5-10元/份 | 集章兑换 |
| 品牌帆布袋 | 15-25元/份 | 优秀投票者抽奖 |
| 定制T恤 | 30-50元/份 | 特别贡献奖 |

## 第四步：失物招领处理

### 失物招领流程

```
拾获物品 → 登记失物信息 → 存放至服务台 → 游客认领 → 核实身份 → 签收确认
```

| 001 | | | | | | | |

### 失物处理规范

- **一般物品**（水杯、雨伞等）：活动结束后保留7天，逾期处理
- **贵重物品**（手机、钱包、证件）：立即报告总控，妥善保管，活动后保留30天
- **危险物品**：立即报告安保人员处理

## 第五步：医疗急救点设置

### 急救点配置标准

| 项目 | 配置要求 |
|------|---------|
| 位置 | 靠近主通道，标识醒目，门口无障碍 |
| 面积 | 至少 10 m²，可放置一张急救床 |
| 人员 | 至少1名持急救证人员值守 |
| 设备 | 急救箱、AED（自动体外除颤器）、血压计 |

### 急救箱标准配置清单

- [ ] 创可贴（大/小各1盒）
- [ ] 纱布卷 5卷
- [ ] 医用胶带 2卷
- [ ] 碘伏棉签 1盒
- [ ] 医用酒精 1瓶
- [ ] 一次性手套 1盒
- [ ] 三角巾 2条
- [ ] 冰袋 4个
- [ ] 体温计 2支
- [ ] 常用药品（退烧药、止痛药、抗过敏药）
- [ ] 急救手册 1本

### 急救响应流程

```
发现伤员/患者
    ↓
立即呼叫急救点志愿者（对讲机/电话）
    ↓
急救人员携带急救箱到场
    ↓
评估伤情：
  - 轻微（擦伤、头晕）→ 现场处理 → 记录
  - 中度（扭伤、流血）→ 转移至急救点处理 → 记录
  - 重度（意识不清、骨折）→ 拨打120 → 陪同送医 → 通知家属
    ↓
填写急救记录表
```

## 常见问题

**Q1：签到排长队怎么办？**
A：①增开签到台；②安排志愿者提前让游客准备好电子票；③对于VIP启动快速通道；④在排队区安排志愿者解答常见问题，减少签到台咨询时间。

**Q2：游客迷路了怎么办？**
A：引导志愿者应主动观察面露困惑的游客。每个区域入口设置清晰的地图指引。可以制作简易的"口袋地图"放在资料袋中。

**Q3：游客在展区拍照是否需要限制？**
A：提前与参赛团队确认是否允许拍摄。不允许拍摄的项目设置"禁止拍照"标识，志愿者礼貌提醒。

**Q4：有人在现场发生冲突怎么办？**
A：立即将当事人带离现场至安静区域，由2名工作人员分别了解情况。轻微纠纷现场调解，严重冲突报告安保并拨打110。

**Q5：遇到极端天气（暴雨/高温）游客接待怎么办？**
A：提前准备雨棚/遮阳棚。签到区备有雨伞借用。高温天准备防暑降温物资（藿香正气水、饮用水）。室内等候区提前开放。

## 历史教训

- 历史活动活动通常为期2-3天，最后一天为对外开放的Demo Day
- 游客签到建议安排在开场前1小时，避免与选手签到时间重叠
- 历史活动的展示区通常设置"项目海报+实物Demo+团队介绍"三要素
- 互动环节中"投票墙"是最受欢迎的形式，游客用贴纸投票选出"最受欢迎项目"
- 急救点建议与场地物业合作，借用其现有急救资源
- 历史教训表明：引导志愿者比标识牌更有效，关键路口必须安排人员
