# 如何搭建活动官网

> **适用场景**：为黑客松活动搭建官方网站，用于活动宣传、报名引导、信息展示。
> **前置条件**：请先完成 `06_宣发与品牌传播.md`，确认品牌视觉和传播策略。

---

## 第一步：规划官网内容结构

### 核心页面规划

| 页面 | 内容 | 优先级 |
|------|------|--------|
| 首页 | 活动主视觉、倒计时、核心亮点、CTA按钮 | P0 |

| 联系我们 | 联系方式、社群二维码 | P1 |

## 第二步：技术选型

### 方案对比

| 方案 | 技术门槛 | 费用 | 灵活性 | 适合场景 | 开发周期 |
|------|---------|------|--------|---------|---------|
| 静态网站（HTML/CSS） | 中 | 域名+托管费 | 高 | 有前端开发能力的团队 | 3-5天 |
| GitHub Pages + 模板 | 低 | 免费（需域名费） | 中 | 预算有限、有基础技术能力 | 1-2天 |
| WordPress | 低 | 域名+主机+主题费 | 中 | 内容管理需求高 | 2-3天 |
| Webflow | 低 | 免费-付费计划 | 高 | 设计要求高、无开发人员 | 2-4天 |
| Notion + Super.so | 极低 | 免费-付费 | 低 | 快速上线、信息展示为主 | 半天 |
| 腾讯云官网建站 | 极低 | 按年付费 | 低 | 完全无技术背景 | 1天 |

### 推荐方案：新手首选

**方案A：GitHub Pages + 模板**（零成本）

```bash
# 步骤：
1. 注册 GitHub 账号
2. 创建仓库：{username}.github.io
3. 选择模板（推荐 Jekyll 主题或 Hugo 主题）
4. 修改配置文件和内容
5. 推送代码，自动部署
6. 绑定自定义域名（可选）
```

**方案B：Webflow**（设计优先）

```
1. 注册 Webflow 账号
2. 选择活动类模板或从零开始
3. 拖拽式编辑页面
4. 添加交互和动画
5. 发布到 Webflow 子域名或绑定自定义域名
```

**方案C：Notion + Super.so**（最快上线）

```
1. 在 Notion 创建活动页面
2. 注册 Super.so 账号
3. 将 Notion 页面绑定到 Super.so
4. 设置自定义域名
5. 自动获得一个简洁的官网
```

## 第三步：设计规范

### 品牌视觉要素

| 要素 | 规范 | 说明 |
|------|------|------|
| 主色 | 根据活动品牌色设定 | 用于按钮、强调文字 |
| 辅色 | 主色的浅色变体 | 用于背景、卡片 |
| 字体（中文） | 思源黑体/阿里巴巴普惠体 | 免费商用字体 |
| 字体（英文） | Inter/Montserrat | 现代感强 |
| 正文字号 | 16px（移动端14px） | 保证可读性 |
| 行高 | 1.6-1.8 | 中文排版舒适 |
| 圆角 | 8px-12px | 统一的圆角风格 |

### 响应式断点

| 设备 | 宽度范围 | 布局调整 |
|------|---------|---------|
| 手机 | < 768px | 单列布局，汉堡菜单 |
| 平板 | 768px - 1024px | 两列布局 |
| 桌面 | > 1024px | 完整布局 |

### 设计检查清单

- [ ] 所有图片有 alt 文本
- [ ] 颜色对比度符合 WCAG AA 标准
- [ ] 按钮最小点击区域 44×44px
- [ ] 所有文字在背景上清晰可读
- [ ] 移动端导航可正常使用
- [ ] 加载时间 < 3秒（首页）

## 第四步：SEO优化

### 基础SEO设置

| 项目 | 内容 | 示例 |
|------|------|------|
| 页面标题 | 活动名称 + 核心关键词 | "活动2026 - 48小时黑客松挑战赛" |
| Meta描述 | 150字以内的活动简介 | "活动2026是面向大学生的48小时黑客松..." |
| 关键词 | 活动相关关键词 | "黑客松, hackathon, 编程比赛, 历史活动" |
| OG图片 | 活动主视觉图（1200×630px） | 用于社交媒体分享预览 |
| OG标题 | 与页面标题一致 | |
| OG描述 | 与Meta描述一致 | |

### SEO检查清单

- [ ] 每个页面有唯一的 title 和 description
- [ ] 图片经过压缩（WebP格式优先）
- [ ] 使用语义化 HTML 标签（h1/h2/h3正确嵌套）
- [ ] 生成 sitemap.xml
- [ ] 配置 robots.txt
- [ ] 添加结构化数据（Event Schema）
- [ ] 页面加载速度优化（目标 < 3秒）
- [ ] 移动端友好（通过 Google Mobile-Friendly Test）

## 第五步：域名备案流程

### 域名注册

| 步骤 | 操作 | 备注 |
|------|------|------|
| 1 | 选择域名注册商 | 推荐：阿里云、腾讯云、Namesilo |
| 2 | 查询域名可用性 | 优先 .com / .cn / .dev |
| 3 | 购买域名 | .cn 域名需要实名认证 |
| 4 | 配置DNS解析 | 指向托管服务器 |

### ICP备案流程（中国大陆服务器必须）

```
第一步：准备材料
  - 主办单位证件（营业执照/组织机构代码证/身份证）
  - 网站负责人身份证正反面
  - 网站负责人幕布照片（接入商提供）
  - 域名证书

第二步：选择接入商提交备案
  - 阿里云：https://beian.aliyun.com
  - 腾讯云：https://cloud.tencent.com/product/ba
  - 华为云：https://www.huaweicloud.com/product/ba

第三步：接入商初审（1-3个工作日）

第四步：工信部短信核验

第五步：管局审核（各省不同，通常7-20个工作日）

第六步：备案通过，获取备案号
  - 在网站页脚添加备案号：京ICP备XXXXXXXX号
  - 链接到 https://beian.miit.gov.cn
```

### 免备案方案

如果不想走备案流程，可以选择：
- 使用海外服务器（GitHub Pages、Vercel、Netlify）
- 使用 Cloudflare Pages
- 使用 Notion + Super.so 等第三方托管

## 常见问题

**Q1：完全没有技术基础，用什么方案最好？**
A：推荐 Notion + Super.so 方案，只需会用 Notion 即可，半小时即可上线。其次是 Webflow，拖拽式编辑，无需代码。

**Q2：域名怎么选？**
A：优先选择 .com 域名，简短好记。活动相关可以考虑 .dev、.hack、.tech 等。.cn 域名需要实名认证。

**Q3：网站需要备案吗？**
A：如果使用中国大陆服务器，必须备案。使用海外服务器（GitHub Pages、Vercel等）则不需要备案。

**Q4：官网多久能搭建完成？**
A：取决于方案和内容准备情况。Notion方案半天，Webflow方案2-4天，自定义开发3-7天。建议在活动前至少2周完成官网。

**Q5：如何让官网被搜索引擎收录？**
A：①提交 sitemap 到百度站长平台和 Google Search Console；②在社交媒体分享官网链接；③确保页面有优质内容和正确的SEO设置。

## 历史教训

- 历史活动官网通常采用简洁现代的单页设计，突出核心信息
- 首页 Hero 区域建议使用活动宣传视频或动态背景，提升视觉冲击力
- 报名入口应放在最显眼的位置，建议在导航栏固定"报名"按钮
- 历史活动官网通常包含"往届回顾"页面，展示历届优秀项目和照片
- FAQ页面非常重要，可以减少组委会的咨询工作量
- 赞助商Logo墙是吸引潜在赞助商的重要展示位
