# 18 执行团队与现场管理

> 本手册适用于活动执行期间的团队管理、现场调度和应急处置。
> 参考案例：活动主办方 历史活动

---

## 一、核心原则

1. **现场只有一个指挥官**：总协调人拥有现场最高决策权
2. **通讯必须畅通**：所有工作人员保持通讯设备在线
3. **预案先行**：所有可能的突发情况都要有预案
4. **提前到岗**：所有工作人员提前30分钟到岗

---

## 二、现场组织架构

### 2.1 现场指挥体系

```
总协调人（1人）
├── 会务组（签到、嘉宾接待、秩序维护）
├── 场务组（场地布置、物料管理、设备维护）
├── 宣发组（摄影摄像、直播、社交媒体）
├── 技术组（设备调试、线上系统、网络保障）
├── 志愿者组（志愿者管理和调配）
└── 机动组（巡视、应急支援）
```

### 2.2 通讯体系

| 通讯方式 | 用途 | 适用范围 |
|---|---|---|
| 工作群（微信/飞书） | 日常沟通、信息发布 | 全员 |
| 对讲机 | 实时调度、紧急通讯 | 各组组长、关键岗位 |
| 电话 | 紧急联络 | 总协调、应急联系人 |

## 三、现场执行流程

### 3.1 活动前一天（T-1天）

| 时间 | 任务 | 负责人 |
|---|---|---|
| 上午 | 物料清点和打包 | 场务组 |
| 下午 | 物料运送至场地、布置开始 | 场务组 |
| 傍晚 | 设备调试（音响、灯光、屏幕） | 技术组 |
| 晚上 | 全流程彩排 | 总协调 |
| 彩排后 | 最终检查、应急预案确认 | 总协调 |

### 3.2 活动当天（T日）

| 时间 | 任务 | 负责人 |
|---|---|---|
| T-2小时 | 全员到岗、最终检查 | 总协调 |
| T-1小时 | 各组就位、通讯测试 | 各组组长 |
| T-30分钟 | 签到准备、设备最终测试 | 会务组、技术组 |
| T | 活动开始 | 全员 |
| 活动中 | 各司其职、实时沟通 | 全员 |
| 活动结束 | 收尾工作、物料回收 | 场务组 |

### 3.3 每日岗前Briefing

每天上岗前15分钟进行简短briefing：
- 昨日回顾（做得好的、需要注意的）
- 今日安排（关键时间节点、特别事项）
- 人员调整（请假、轮班）
- 应急提醒（天气、设备状态）

## 四、现场管理规范

### 4.1 签到管理

- 签到台位置：入口显眼处
- 签到方式：电子签到（扫码）+ 纸质签到（备份）
- 物料发放：签到时同步发放胸牌、手册等
- 迟到处理：设置迟到签到通道，避免影响已入场人员

### 4.2 嘉宾接待

- VIP通道：设置嘉宾专用签到通道
- 休息室：嘉宾专用休息区域
- 专人对接：每位嘉宾有专人负责引导
- 时间提醒：环节前提醒嘉宾候场

### 4.3 秩序维护

- 入口控制：安排志愿者控制人流
- 区域隔离：不同身份区域用围栏/标识隔离
- 巡视制度：机动组定时巡视全场
- 紧急出口：确保所有紧急出口畅通

### 4.4 设备管理

- 设备清单：所有设备编号登记
- 使用记录：设备借出和归还记录
- 故障处理：设备故障时的快速响应流程
- 备用设备：关键设备准备备用方案

## 五、应急预案

### 5.1 常见突发情况

| 突发情况 | 预案 | 负责人 |
|---|---|---|
| 设备故障 | 备用设备替换、技术人员快速维修 | 技术组 |
| 网络中断 | 备用热点、离线方案 | 技术组 |
| 停电 | 备用发电机、应急照明 | 场务组 |
| 人员受伤 | 急救箱、120急救电话 | 总协调 |
| 人流超载 | 暂停入场、分流引导 | 会务组 |
| 天气变化 | 雨棚、室内备用方案 | 场务组 |
| 嘉宾缺席 | 备选嘉宾、调整流程 | 会务组 |
| 消防安全 | 疏散路线、灭火器 | 总协调 |

### 5.2 应急响应流程

```
发现异常 → 判断严重程度
├── 轻微：现场处理，记录备案
├── 一般：上报组长，按预案处理
└── 严重：立即上报总协调，启动应急预案
```

## 七、易遗漏事项

1. **工作人员餐饮**：工作人员的餐饮安排容易被忽略
2. **充电设备**：现场需要充足的充电设备和充电宝
3. **垃圾处理**：活动期间的垃圾清理安排
4. **噪音控制**：不同区域的噪音隔离
5. **温度控制**：室内温度的监控和调节
6. **卫生间指引**：卫生间的明确指引
7. **失物招领**：设置失物招领处
8. **工作人员休息**：工作人员需要有轮班休息的时间和空间

## 九、历史教训

> **历史活动 现场管理**：使用执行分工信息文档统一管理所有人员分工。总协调组（总负责人、策划负责人）负责盯紧各组进度、调配人员、应急决策。每个环节都有明确的负责人和执行细则。

> **历史活动 的教训**：现场通讯必须畅通，所有工作人员的电话号码必须提前收集。机动组必须时刻保持至少2-3个人在开发场地内，并且指定每个时刻一个人主负责。理论上所有没有考虑到的事情，机动组都应该去对接帮助负责。
