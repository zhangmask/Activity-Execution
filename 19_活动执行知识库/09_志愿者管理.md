# 09 志愿者管理

> 本手册适用于活动志愿者的招募、培训、分工和管理。
> 参考案例：活动主办方 历史活动

---

## 一、核心原则

1. **做早招募**：志愿者招募不晚于活动前20天
2. **做好分工**：每个志愿者有明确的岗位和职责
3. **做足招待**：志愿者也是团队成员，要给予足够的尊重和回报
4. **专人管理**：志愿者工作分配和现场管理需要有总负责人

---

## 二、志愿者招募

### 2.1 需求分析

根据活动规模确定志愿者人数：

| 活动规模 | 参与人数 | 建议志愿者人数 |
|---|---|---|
| 小型（50人以下） | 30-50人 | 5-10人 |
| 中型（50-200人） | 50-200人 | 15-30人 |
| 大型（200人以上） | 200-500人 | 30-60人 |

### 2.2 岗位设置

| 岗位 | 职责 | 人数建议 |
|---|---|---|
| 签到引导 | 签到、发放物料、指引 | 2-3人 |
| 场务 | 物料搬运、场地布置 | 4-6人 |
| 引导 | 路线指引、咨询 | 3-5人 |
| 机动 | 支援各组、应急 | 3-5人 |
| 摄影协助 | 协助摄影摄像 | 1-2人 |
| 技术协助 | 设备调试、线上支持 | 1-2人 |

### 2.3 招募渠道

- 高校社团/学生会
- 社交媒体招募帖
- 往届志愿者推荐
- 合作方推荐

### 2.4 招募信息模板

```
【活动名称】志愿者招募
【活动时间】xx月xx日 - xx月xx日
【活动地点】xxx
【招募人数】xx人
【岗位说明】
  - 签到引导：xxx
  - 场务：xxx
  - ...
【志愿者权益】
  - 志愿者证书
  - 活动周边礼品
  - 餐饮供应
  - 志愿时长认证
【报名方式】填写报名表：xxx链接
【截止日期】xx月xx日
```

## 三、志愿者培训

### 3.1 培训内容

| 内容 | 说明 |
|---|---|
| 活动概况 | 活动背景、规模、流程 |
| 岗位职责 | 具体工作内容和标准 |
| 场地熟悉 | 场地布局、各区域功能 |
| 应急处理 | 突发情况的处理流程 |
| 通讯方式 | 对讲机/群聊使用、紧急联系人 |
| 行为规范 | 着装要求、服务态度、保密要求 |

### 3.2 培训时间

- **线上培训**：活动前3-5天，通过视频会议进行
- **现场培训**：活动前一天，到场实地熟悉
- **岗前 briefing**：每天上岗前15分钟

---

## 四、志愿者权益

| 权益 | 说明 |
|---|---|
| 志愿者证书 | 活动后颁发，可作为社会实践证明 |
| 周边礼品 | 活动专属周边一份 |
| 餐饮供应 | 活动期间的工作餐 |
| 社交机会 | 与嘉宾、选手交流的机会 |
| 志愿时长 | 认证志愿时长（如适用） |
| 聚餐交流 | 活动后的志愿者聚餐 |

---

## 五、现场管理

### 5.1 排班制度

- 每个岗位设置组长，负责本组人员管理
- 安排轮班制度，确保每人有休息时间
- 机动组随时待命，支援工作量过大的岗位

### 5.2 通用工作要求

1. 所有志愿者**提前30分钟到岗**，做好岗前准备
2. 不得迟到、早退、旷工，特殊情况需提前报备
3. 穿着统一的志愿者服装/标识
4. 保持手机畅通，及时回复群消息
5. 遇到无法处理的问题，立即上报组长

### 5.3 通讯管理

- 建立志愿者微信群，实时沟通
- 关键岗位配备对讲机
- 制作现场通讯录，包含所有工作人员电话
- 设置紧急联系人（总协调组）

---

## 六、易遗漏事项

1. **志愿者筛选**：不要来者不拒，要筛选有责任心、时间充裕的人
2. **工作时间冲突**：提前收集每个人可参与的时间段
3. **志愿者服装**：统一服装或标识（如手环、贴纸），方便识别
4. **休息区域**：志愿者需要有休息和存放个人物品的区域
5. **餐饮安排**：志愿者的餐饮不要遗漏
6. **交通补贴**：如果志愿者需要远距离通勤，考虑交通补贴
7. **感谢信**：活动结束后发送感谢信，维护长期关系
8. **备用人员**：关键岗位需要有备用人员，应对临时缺席

## 八、历史教训

> **历史活动 志愿者管理**：使用飞书报名表收集志愿者信息。志愿者分为签到、设计、物料、引导、机动等组别。机动组执行手册单独文档，明确巡视和应急职责。

> **历史活动 的教训**：务必重视志愿者工作，做早招募、做好分工、做足招待。关键岗位需要2-3个人轮替，避免一个人从头扛到尾。
