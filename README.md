# 活动执行知识库 Activity Execution Knowledge Base

> 一套完整的活动执行知识库，涵盖从策划到复盘的全流程，包含18个模块、41个操作指南、60个模板，让新手也能照着办好一场大型活动。

---

## 核心能力

| 能力 | 说明 |
|------|------|
| **快速上手** | 新手看完对应模块的操作指南，就能执行具体任务 |
| **避免踩坑** | 每个模块都有"历史教训"，告诉你前人踩过的坑 |
| **标准化流程** | 提供时间线、检查清单、模板，减少遗漏 |
| **灵活组合** | 18个模块可独立使用，也可组合成完整方案 |

## 适用场景

- 黑客松（Hackathon）
- Demo Day / 创业集市
- 技术大会 / 行业峰会
- 社区活动 / 线下聚会
- 任何需要多人协作的大型活动

## 快速开始

### 如果你只有1小时

1. 读 `00_总索引.md`（10分钟）
2. 读你负责模块的主文档（20分钟）
3. 读对应的子操作文档（30分钟）

### 如果你只有1天

1. 读 `00_总索引.md`
2. 读 `00_角色映射表.md` + `00_变量与占位符说明.md`
3. 读 `01_事务统筹与团队分工.md` + `1_1_如何从零组建活动团队.md`
4. 读 `99_全部待办清单.md`
5. 根据你的角色，读2-3个核心模块

### 如果你要完整学习

按编号顺序读完所有文档，预计需要4-6小时。

## 核心原则

1. **时间线管理** - 以T日（活动日）为基准倒推，T-60天开始筹备
2. **分工明确** - 每件事有且只有一个第一责任人
3. **文档先行** - 所有决策留文字记录，使用在线协作文档
4. **预留冗余** - 时间预留20%缓冲，物料多准备20%备用

---

## 使用方式

### 1. 填写角色映射

打开 `00_角色映射表.md`，在"映射人名"列填写实际负责人姓名。

### 2. 填写变量值

打开 `模板/44_变量值维护表.csv`，填写活动名称、场地、时间等变量值。

### 3. 按阶段执行

参考 `00_总索引.md` 中的"按阶段使用"指南，逐步完成各阶段任务。

### 4. 使用模板

在 `模板/` 文件夹中找到对应模板，填写使用。
